## Строгий режим

```js
// ❌ Плохо - создаётся глобальная переменная
age = 10;

// ✅ Хорошо - используйте строгий режим
'use strict'; // прописывать в начале каждого файла JS
```

**Строгий режим** запрещает создавать переменные без `let`/`const`/`var`

```js
"use strict";

// use strict не даст жэтому коду скомпилироваться, т.к. будет ошибка SyntaxError: Dublicate parameter name note allowed in this context 
function exmp(a,a){
	return a;
}
```

- полезная шутка - обезопасивает код, рекоменудется использвоать
- в реакте все устроено по дургому, там вручную эта штука не прописывается

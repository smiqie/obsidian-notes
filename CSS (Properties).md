# `box-sizing`

```css
/* –†–∞–∑–º–µ—Ä—ã –ù–ï–ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã: width/height —É—á–∏—Ç—ã–≤–∞—é—Ç —Ç–æ–ª—å–∫–æ –∫–æ–Ω—Ç–µ–Ω—Ç.
   –ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ padding –∏–ª–∏ border —ç–ª–µ–º–µ–Ω—Ç —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –±–æ–ª—å—à–µ. */

/* –†–∞–∑–º–µ—Ä—ã –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã: width/height –≤–∫–ª—é—á–∞—é—Ç –∫–æ–Ω—Ç–µ–Ω—Ç + padding + border.
   –ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ –æ—Ç—Å—Ç—É–ø–æ–≤ –∏–ª–∏ –≥—Ä–∞–Ω–∏—Ü –æ–±—â–∏–π —Ä–∞–∑–º–µ—Ä –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è. */

```
1. `box-sizing` –≤–ª–∏—è–µ—Ç —Ç–æ–ª—å–∫–æ –Ω–∞ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ —Ä–∞–∑–º–µ—Ä—ã —ç–ª–µ–º–µ–Ω—Ç–∞: `content`, `padding`, `border`

# `position`, `top`, `left`, `bottom`, `right`

```css
/* –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é - –æ–±—ã—á–Ω—ã–π –ø–æ—Ç–æ–∫ –¥–æ–∫—É–º–µ–Ω—Ç–∞ */

/* —Å–¥–≤–∏–≥–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –µ–≥–æ –æ–±—ã—á–Ω–æ–≥–æ –ø–æ–ª–æ–∂–µ–Ω–∏—è */

top: 10px;   /* –≤–Ω–∏–∑ –Ω–∞ 10px */
left: 20px;  /* –≤–ø—Ä–∞–≤–æ –Ω–∞ 20px */

/* –≤—ã—Ä—ã–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –ø–æ—Ç–æ–∫–∞, –ø–æ–∑–∏—Ü–∏–æ–Ω–∏—Ä—É–µ—Ç—Å—è –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –±–ª–∏–∂–∞–π—à–µ–≥–æ —Ä–æ–¥–∏—Ç–µ–ª—è —Å position != static */
position: absolute;
top: 0;

/* —Ñ–∏–∫—Å–∏—Ä—É–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –æ–∫–Ω–∞ –±—Ä–∞—É–∑–µ—Ä–∞ */
position: fixed;
top: 0;

/* –∫–æ–º–±–∏–Ω–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –ø–æ–≤–µ–¥–µ–Ω–∏–µ relative + fixed, –ø—Ä–∏–ª–∏–ø–∞–µ—Ç –ø—Ä–∏ —Å–∫—Ä–æ–ª–ª–µ –≤ –ø—Ä–µ–¥–µ–ª–∞—Ö —Ä–æ–¥–∏—Ç–µ–ª—è */

```

1. `position`

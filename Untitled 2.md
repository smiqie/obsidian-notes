# Логические операторы в JavaScript

Логические операторы используются для проверки условий и работы с логическими значениями. В JS они также обладают особенностью "короткого замыкания" (short-circuiting).

---

## 1. `||` (ИЛИ)
Возвращает **первое истинное (truthy)** значение или последнее, если все ложны.

```javascript
const result = '' || null || false || 0 || 'Hi';
console.log(result); // 'Hi' (первое truthy значение)

const allFalse = '' || null || 0;
console.log(allFalse); // 0 (последнее значение, так как истинных нет)
```

**Алгоритм работы:**
1. Вычисляет операнды слева направо.
2. Преобразует каждый операнд в логический тип.
3. Если результат `true`, останавливается и возвращает **исходное** значение этого операнда.
4. Если все операнды ложны, возвращает последний.

**Примеры:**
```javascript
false || true           // true
0 || 100               // 100
'' || 'Hello'          // 'Hello'
null || undefined || 5 // 5
```

---

## 2. `&&` (И)
Возвращает **первое ложное (falsy)** значение или последний операнд, если все истинны.

```javascript
const result = 'Hi' && true && 5 && null;
console.log(result); // null (первое falsy значение)

const allTrue = 1 && 2 && 3;
console.log(allTrue); // 3 (все истинны, возвращен последний)
```

**Алгоритм работы:**
1. Вычисляет операнды слева направо.
2. Если результат `false`, останавливается и возвращает **исходное** значение этого операнда.
3. Если все операнды истинны, возвращает последний.

> [!TIP]
> **Приоритет:** У `&&` приоритет выше, чем у `||`. Выражение `a || b && c` выполнится как `a || (b && c)`.

---

## 3. `??` (Nullish Coalescing / Нулевое слияние)
Возвращает правый операнд, только если левый равен `null` или `undefined`. В остальных случаях возвращает левый.

```javascript
const count = 0 ?? 10;    // 0 (т.к. 0 это не null/undefined)
const name = null ?? 'Guest'; // 'Guest'
```

**Разница с `||`:**
- `||` возвращает правое значение для любого "ложного" (falsy) значения: `0`, `""`, `false`, `null`, `undefined`.
- `??` возвращает правое значение **только** для `null` и `undefined`. Это полезно, когда `0` или `""` являются валидными данными.

---

## 4. `!` (НЕ) и `!!` (Двойное НЕ)
Оператор `!` инвертирует логическое значение.

```javascript
!true  // false
!0     // true (0 -> false, инверсия -> true)
```

**Двойное НЕ (`!!`)** используется для явного приведения значения к логическому типу (аналог `Boolean(value)`):
```javascript
!!'text' // true
!!0      // false
```

---

## 5. Тернарный оператор `? :`
Единственный оператор, принимающий три операнда. Является краткой записью `if...else`.

**Синтаксис:** `условие ? значение_если_true : значение_если_false`

```javascript
const age = 20;
const access = age >= 18 ? 'Allowed' : 'Denied';
```

---

## Практическое применение: Short-circuit evaluation

1. **Значения по умолчанию (через `||` или `??`):**
   ```javascript
   function greet(name) {
     const userName = name || 'Stranger';
     console.log(`Hello, ${userName}!`);
   }
   ```

2. **Выполнение кода по условию (через `&&`):**
   ```javascript
   // Вместо if (isLoggedIn) { showProfile(); }
   isLoggedIn && showProfile();
   ```

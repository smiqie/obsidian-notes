# Пути к файлам

## Относительные пути - относительно текущего файла

```html
<!-- Явно из текущей папки (рекомендуется) -->
<img src="./img/example.png" alt="">

<!-- То же самое, без "./" -->
<img src="img/example.png" alt="">

<!-- Выход на уровень выше -->
<img src="../img/example.png" alt="">
```

## Абсолютные пути - относительно корня проекта

```html
<!-- Всегда начинаются с "/" -->
<img src="/img/example.png" alt="">
<a href="/pages/about.html">О нас</a>
```

## Полные URL - внешние ресурсы

```html
<img src="https://example.com/img/photo.jpg" alt="">
<a href="https://google.com">Google</a>
```

- 💡 Лучше ссылаться на локальные файлы напрямую, а не через URL — так файлы не потеряются при переезде

# `<img>` vs `background-image`

|                      | `<img>`              | `background-image`        |
|----------------------|----------------------|---------------------------|
| Часть контента       | ✅ Да                | ❌ Нет                    |
| Для оформления       | ❌ Нет               | ✅ Да                     |
| SEO                  | ✅ Учитывается       | ❌ Не индексируется       |
| Управление через CSS | 🔸 Ограничено        | 🔹 Полный контроль       |
| Примеры              | Логотипы, фото товаров, превью | Фоны, декоративные элементы |

---

# Форматы изображений

- `jpg` / `jpeg` - фотографии, полезный для пользователя контент (изображение товара, превью к новости и подобное)
- `png` - изображения с прозрачным фоном
- `svg` - векторная графика (иконки, логотипы, декор)
	- не теряет качество при масштабировании, весит меньше растровых форматов

# Шрифты

**Шрифт** - набор символов определённого начертания и стиля, определяющий визуальный облик текста

- **Serif (с засечками)** - шрифты с небольшими «хвостиками» на концах букв
	- используются в печатных изданиях, книгах, для классического/формального стиля
- **Sans-serif (без засечек)** - шрифты с чистыми, ровными линиями
	- используются в веб-дизайне (лучше читаются на экранах), для современного/минималистичного стиля

## Иконочные шрифты

**Иконочные шрифты** - шрифты, где вместо букв находятся иконки и символы (стрелки, логотипы, значки соцсетей и т.д.)

Преимущество: редактируются как обычный текст через CSS:

```css
.icon {
  font-size: 24px;
  color: #ff0000;
  font-weight: bold;
}
```

# Структура проекта

```
src/           ← исходные материалы
├── css/       ← стили
├── scss/      ← scss стили
├── js/        ← скрипты
├── fonts/     ← шрифты
├── img/       ← растровые изображения
├── icons/     ← векторные изображения (svg)
├── index.html ← главная страница
└── *.html     ← остальные страницы

dist/          ← готовая сборка (distribution)
```

- `src` - черновик, `dist` - финальная оптимизированная версия
- `index.html` - стандартное имя главной страницы, браузеры открывают её первой
- 💡 Названия папок и файлов писать строчными буквами через дефис (`my-site`, `main-page.html`) - некоторые серверы чувствительны к регистру

# Хорошие практики

- Одиночные теги лучше закрывать - читается понятнее:

```html
<!-- хуже -->
<input>

<!-- лучше -->
<input />
```

- Интерактивные элементы не должны содержать блочные элементы:

```html
<!-- неправильно -->
<button>
  <p>Текст</p>
</button>

<!-- правильно -->
<button>Текст</button>
```

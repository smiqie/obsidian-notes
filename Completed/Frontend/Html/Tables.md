# `table` - таблица

```html
<table>
  <caption>Продажи за 2024 год</caption>

  <thead>
    <tr>
      <th>Месяц</th>
      <th>Продажи</th>
      <th>Прибыль</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Январь</td>
      <td>1000</td>
      <td>500</td>
    </tr>
    <tr>
      <td>Февраль</td>
      <td>1200</td>
      <td>600</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td>Итого</td>
      <td>2200</td>
      <td>1100</td>
    </tr>
  </tfoot>
</table>
```

- `<table>` - добавляет таблицу ✅ a11y
- `<caption>` - заголовок таблицы ✅ a11y (может быть только один)
- `<thead>` - обёртка верхней части таблицы (заголовки столбцов) ✅ a11y
- `<tbody>` - основная часть контента ✅ a11y
- `<tfoot>` - нижняя часть (итоговые подсчёты, суммы) ✅ a11y
- `<tr>` - строка в таблице
- `<th>` - заголовочная ячейка ✅ a11y
- `<td>` - обычная ячейка

# Объединение ячеек

```html
<table>
  <tr>
    <th colspan="2">Объединённый заголовок</th> <!-- 2 ячейки по горизонтали -->
  </tr>
  <tr>
    <td rowspan="2">Высокая ячейка</td> <!-- 2 ячейки по вертикали -->
    <td>Обычная ячейка</td>
  </tr>
  <tr>
    <td>Ещё ячейка</td>
  </tr>
</table>
```

- `colspan` - объединяет ячейки по горизонтали (указывает количество столбцов)
- `rowspan` - объединяет ячейки по вертикали (указывает количество строк)

# Шаблон HTML-документа

|Тег / элемент|Описание|
|---|---|
|`<!DOCTYPE html>`|Объявляет тип документа и версию HTML5|
|`<html lang="ru">`|Корневой элемент страницы, атрибут `lang` указывает язык|
|`<head>`|Метаданные страницы: кодировка, подключение стилей и скриптов, SEO-настройки|
|`<title>`|Заголовок страницы (отображается во вкладке браузера и результатах поиска)|
|`<body>`|Основное содержимое страницы: текст, изображения, кнопки, блоки и т.д.|

# Мета-теги

Мета-теги - технические теги, которые определяют параметры страницы для браузеров, поисковиков и устройств. Прописываются внутри `<head>`

```html
<!-- Кодировка страницы (UTF-8 поддерживает кириллицу, латиницу, эмодзи) -->
<meta charset="UTF-8">

<!-- Описание страницы в поисковой выдаче -->
<meta name="description" content="Описание страницы">

<!-- Адаптивность на мобильных устройствах -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

- `charset` - кодировка, всегда `UTF-8`
- `description` - отображается под заголовком в поисковой выдаче
	- рекомендуемая длина: до 125 символов на мобильных, до 150 на ПК - всё лишнее обрезается
- `viewport` - обязателен для адаптивной вёрстки

# Подключение стилей

## Через `<link>` (внешний файл) - рекомендуется

```html
<link rel="stylesheet" href="style.css">

<!-- CSS-файл не обязательно должен лежать рядом с HTML -->
<link rel="stylesheet" href="../css/style.css">
```

- `rel="stylesheet"` - говорит браузеру, что это таблица стилей
- `href` - путь к файлу
	- `../` - переход на уровень выше в файловой структуре

## Через `<style>` (встроенные стили)

```html
<head>
  <style>
    p { color: blue; }
    .box { background-color: red; }
  </style>
</head>
```

- Пишется в `<head>`, чтобы стили применились до отображения элементов
- Не рекомендуется - лучше выносить CSS в отдельный файл

## Через атрибут `style` (инлайн-стили)

```html
<p style="color: red; font-size: 16px;">Текст</p>
```

- Не рекомендуется - засоряет разметку и нарушает разделение HTML и CSS

# Подключение скриптов

```html
<!-- В конце <body> — рекомендуется -->
<body>
  ...
  <script src="script.js"></script>
</body>
```

- Тег `<script>` прописывается в конец `<body>`, чтобы браузер сначала отрисовал страницу, а потом загрузил JS

# Фавиконка

```html
<link rel="icon" href="favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="icons/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="icons/favicon-16x16.png">
```

- Иконка сайта во вкладке браузера
- Для полной кросс-платформенной поддержки нужно несколько размеров

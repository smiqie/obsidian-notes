# `scroll-behavior` - поведение прокрутки

```css
/* для всей страницы */
html {
  scroll-behavior: smooth;
}

/* для конкретного контейнера */
.scrollable-container {
  scroll-behavior: smooth;
  overflow-y: scroll;
}
```

- определяет, как будет вести себя анимация скролла после нажатия на якорную ссылку или после программного скролла через JS
- **Важно:** работает только для программной прокрутки (якоря, JavaScript) - не влияет на ручную прокрутку колесиком или тачпадом
- За прокрутку всей страницы отвечает элемент `html` - свойство применяется к нему
- **Значения:**
    - `auto` - мгновенная прокрутка (по умолчанию)
    - `smooth` - плавная анимированная прокрутка

# `cursor` - внешний вид курсора

```css
cursor: auto;
cursor: pointer;
cursor: none;
cursor: url('cat.png'), auto; /* кастомный курсор с fallback */
```

- меняет внешний вид курсора при наведении на элемент
- **Популярные значения:**
    - `auto` - браузер сам выбирает курсор (стрелка, рука для ссылок, черта для инпутов) (по умолчанию)
    - `default` - стандартная стрелка
    - `none` - курсор скрыт
    - `text` - вертикальная черта (для текста)
    - `pointer` - рука с указательным пальцем ==!==
- **Остальные значения:**
    - `grab` / `grabbing` - рука (возможность перетащить / уже тащит)
    - `move` - стрелки во все стороны (можно двигать элемент)
    - `crosshair` - крест
	    - обычно добавляют для изображений, в которых можно выделить опредленную область
    - `zoom-in` / `zoom-out` - лупа увеличить/уменьшить
    - `copy` - стрелка с зелёным плюсиком
    - `not-allowed` / `no-drop` - перечёркнутый круг (действие запрещено / нельзя перетащить)
	    - `not-allowed` такой же как и `no-drop` , но на других ОС будет выглядеть чутка иначе
    - `progress` / `wait` - кружок загрузки
    - `help` - знак вопроса
    - `alias` - загнутая стрелка 
	    - так иногда обозначают ссылки ведущие на сторонний сайт
    - `cell` - выбор ячейки таблицы
    - `col-resize` / `row-resize` - изменение размера колонки/строки таблицы
    - `context-menu` - стрелка с иконкой списка (контекстное меню)
	    - обычно добавляют на меню открывающиеся при клике на ПКМ
    - `vertical-text` - горизонтальная черта (вертикальный текст)
    - `all-scroll` - скролл во все стороны
    - ![[cursor.png]]
- **Кастомный курсор:** `cursor: url('cat.png'), auto;` - PNG или SVG, не более 32×32px, обязательно указывать fallback-значение

# `user-select` - выделение текста

```css
user-select: auto;
user-select: none;
```

- определяет, можно ли выделить текст в элементе
- **Значения:**
    - `auto` - браузер сам определяет правила, в большинстве случаев текст выделяется (по умолчанию)
    - `none` - отключает выделение текста
	    - полезно для интерактивных элементов (кнопки и т.д.), чтобы при клике текст не выделялся случайно

# `pointer-events` - возможность нажатия

```css
pointer-events: auto; /* по умолчанию */
pointer-events: none; /* нельзя нажать */
```

```html
<div class="box">
  <button>Click</button>
</div>
```

```css
.box {
  pointer-events: none; /* применится и к детям */
}
button {
  pointer-events: auto; /* переопределяем, чтобы кнопка работала */
}
```

- отвечает за то, может ли пользователь взаимодействовать с элементом
- **Важно:** свойство наследуется всеми дочерними элементами
- **Значения:**
    - `auto` - взаимодействие разрешено (по умолчанию)
    - `none` - взаимодействие отключено (клики, hover и т.д. не работают)
- Выручает когда интерактивный элемент вложен в другой и нужно точечно отключить взаимодействие с одним из слоёв

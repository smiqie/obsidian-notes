# `transition` - плавность переходов

```css
/* шорткат (значения по умолчанию) */
transition: all 0s ease 0s; /* property duration timing-function delay */

/* можно не указывать всё */
transition: 1s; /* только duration, остальное по умолчанию */ 
transition: 1s 2s; /* duration delay */

/* подсвойства */
transition-property: all;
transition-duration: 0s;
transition-timing-function: ease;
transition-delay: 0s;
```

```css
/* несколько переходов */
transition:
  transform .2s ease 0s,
  font-size .2s ease 0s;
```

- добавляет плавность при изменении CSS-свойств элемента
- **Важно:** задавать `transition` нужно элементу в **статичном состоянии** (не в `:hover`)
- Работает не только с трансформациями, но и с другими свойствами (`color`, `box-shadow`, `filter`, `opacity`, `top`, `right` и тд.)
- Сокращённые варианты записи: `transition: 1s;` или `transition: 1s 2s;` - остальные подсвойства примут значения по умолчанию

---

## Подсвойства

1. `transition-property` - CSS-свойство, на которое влияет переход
    - `all` - применяется ко всем изменяемым свойствам (по умолчанию)
    - или конкретное свойство: `transform`, `color`, `opacity` и т.д.

2. `transition-duration` - длительность перехода
    - Значения в `s` или `ms`

3. `transition-delay` - длительность задержки перед началом перехода
    - Значения в `s` или `ms

4. `transition-timing-function` - функция скорости изменения свойства
    - **Значения:**
        - `ease` - медленно → быстро → медленно (по умолчанию)
        - `ease-in` - начинает медленно, заканчивает быстро
        - `ease-out` - начинает быстро, заканчивает медленно
        - `ease-in-out` - медленно → быстро → медленно (симметрично)
        - `linear` - равномерная скорость
        - `cubic-bezier(0.1, 0.6, 1.0, 0.1)` - кастомная кривая из 4 точек
	        - все ключевые слова выше - это шорткаты на основе этой функции

# Хорошая практика

```css
/* Правильно */
transition: all 0.3s;

/* Тоже правильно (можно опустить 0) */
transition: all .3s;
```

- можно опускать `0` в десятичных дробях (`.3s` вместо `0.3s`)

# `transform` - трансформации элемента

```css
transform: none;

/* несколько трансформаций сразу через пробел */
transform: translate(30px, -30px) scale(1.5) skew(20deg, 5deg);
```

- отвечает за трансформации элемента (смещение, масштабирование, поворот, искажение)
- трансформации **не влияют на поток** - соседние элементы всегда видят элемент на его изначальном месте

- **Значения:**
    
    - `none` - трансформации не применены (по умолчанию)
    
    1. **Смещение**
        
        - `translateX(50px)` - смещает по горизонтали
	        - значения могут быть отрицательными (смещение в обратную сторону)
        - `translateY(-3px)` - смещает по вертикали
	        - отрицательное значение - вверх
        - `translate(12px, 45px)` - смещает по двум осям `<x> <y>`

    2. **Масштабирование**
        
        - `scaleX(2)` - масштабирование по оси X
	        - отрицательное значение - зеркалит по горизонтали
        - `scaleY(1.2)` - масштабирование по оси Y
	        - отрицательное значение - зеркалит по вертикали
        - `scale(1.3, -0.5)` - масштабирование по двум осям `<x> <y>` с сохранением пропорций (что в большинстве случаев и нужно)
	        - если значения совпадают - `scale(3)`
        - Значение - коэффициент: `1` = оригинальный размер (по умолчанию)

    3. **Поворот**
        
        - `rotate(45deg)` - поворот элемента по часовой стрелке относительно центра (по умолчанию)
        - `0deg` = не повёрнут, `360deg` = полный оборот
        - Отрицательное значение - против часовой стрелки
        - Единица `turn`: `1turn` = `360deg`, `0.5turn` = `180deg`

    4. **Искажение**
        
        - `skewX(0deg)` - искажение по оси X
        - `skewY(21deg)` - искажение по оси Y
        - `skew(12deg, 31deg)` - по двум осям `<x> <y>`
        - **Значения:** `deg` или `turn` 
	        - могут быть отрицательными

# `transform-origin` - точка трансформации

```css
transform-origin: 50% 50%; /* center - по умолчанию */
transform-origin: 0 0;
transform-origin: top left;
```

- задаёт точку, относительно которой применяются трансформации
- **Паттерн:** `transform-origin: <X> <Y>`
- **Значения:** `center`, `top`, `bottom`, `left`, `right` / `%` / `px` и тд.

# Отдельные свойства трансформации

```css
/* вместо функций внутри transform можно использовать отдельные свойства */
translate: 30px 12px;  /* x y */
translate: 12px;       /* только x */

scale: 1.5 1.2;        /* x y */
scale: 1.5;            /* оба направления одновременно */

rotate: 45deg;
```

- современная альтернатива функциям внутри `transform`
- `translate` и `scale`: первое значение - X, второе - Y
- `scale` с одним значением масштабирует по обеим осям
- `rotate` принимает одно значение

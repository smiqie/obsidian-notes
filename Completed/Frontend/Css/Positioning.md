# `position`, `top`, `left`, `bottom`, `right`, `inset`

```css
/* –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é - –æ–±—ã—á–Ω—ã–π –ø–æ—Ç–æ–∫ –¥–æ–∫—É–º–µ–Ω—Ç–∞ */

/* —Å–¥–≤–∏–≥–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –µ–≥–æ –æ–±—ã—á–Ω–æ–≥–æ –ø–æ–ª–æ–∂–µ–Ω–∏—è */

/* –≤—ã—Ä—ã–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –ø–æ—Ç–æ–∫–∞, –ø–æ–∑–∏—Ü–∏–æ–Ω–∏—Ä—É–µ—Ç—Å—è –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –±–ª–∏–∂–∞–π—à–µ–≥–æ —Ä–æ–¥–∏—Ç–µ–ª—è —Å position != static */

/* —Ñ–∏–∫—Å–∏—Ä—É–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –æ–∫–Ω–∞ –±—Ä–∞—É–∑–µ—Ä–∞ */

/* –∫–æ–º–±–∏–Ω–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –ø–æ–≤–µ–¥–µ–Ω–∏–µ relative + fixed, –ø—Ä–∏–ª–∏–ø–∞–µ—Ç –ø—Ä–∏ —Å–∫—Ä–æ–ª–ª–µ –≤ –ø—Ä–µ–¥–µ–ª–∞—Ö —Ä–æ–¥–∏—Ç–µ–ª—è */

top: 0;
```

- –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –¥—Ä—É–≥–∏—Ö

- **–ó–Ω–∞—á–µ–Ω–∏—è:**
    - `static` - –æ–±—ã—á–Ω—ã–π –ø–æ—Ç–æ–∫ (—ç–ª–µ–º–µ–Ω—Ç —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω –≤ –æ–±—ã—á–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ), `top`/`left` –∏ —Ç.–¥. –Ω–µ —Ä–∞–±–æ—Ç–∞—é—Ç (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é)
    - `relative`
        - —Å–¥–≤–∏–≥–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç **–≤–∏–∑—É–∞–ª—å–Ω–æ** –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ —Å–≤–æ–µ–≥–æ –º–µ—Å—Ç–∞, **–º–µ—Å—Ç–æ –≤ –ø–æ—Ç–æ–∫–µ –æ—Å—Ç–∞—ë—Ç—Å—è –ø—Ä–µ–∂–Ω–∏–º** (—Ç.–µ. —ç–ª–µ–º–µ–Ω—Ç –ø–æ —Å—É—Ç–∏ –æ—Å—Ç–∞—ë—Ç—Å—è –Ω–∞ —Å–≤–æ—ë–º –º–µ—Å—Ç–µ)
	        - –ø–æ —Å—É—Ç–∏ —ç–ª–µ–º–µ–Ω—Ç –ø–æ–∑–∏—Ü–∏–æ–Ω–∏—Ä—É–µ—Ç—Å—è —Ç–∞–∫–∂–µ –∫–∞–∫ –∏ `static`, –Ω–æ –Ω–∞ –Ω–µ–≥–æ –±—É–¥—É—Ç –¥–µ–π—Å—Ç–≤–æ–≤–∞—Ç—å —Å–≤–æ–π—Å—Ç–≤–∞ —Å–º–µ—â–µ–Ω–∏—è
        - —Å–∞–º–æ –ø–æ —Å–µ–±–µ —Å–æ —Å–≤–æ–π—Å—Ç–≤–∞–º–∏ —Å–º–µ—â–µ–Ω–∏—è **–∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –æ—á–µ–Ω—å —Ä–µ–¥–∫–æ**, –Ω–æ **—á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–∞–∫ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä –¥–ª—è `absolute` –¥–µ—Ç–µ–π**
        - –Ω–∏–∫–∞–∫ –Ω–µ –≤–ª–∏—è–µ—Ç –Ω–∞ —Å–æ—Å–µ–¥–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã
    - `absolute`
        - **–≤—ã–±–∏–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –æ—Å–Ω–æ–≤–Ω–æ–≥–æ –ø–æ—Ç–æ–∫–∞** (—ç–ª–µ–º–µ–Ω—Ç –ø–µ—Ä–µ—Å—Ç–∞—ë—Ç –∑–∞–Ω–∏–º–∞—Ç—å —Å–≤–æ—ë –ø–µ—Ä–≤–æ–Ω–∞—á–∞–ª—å–Ω–æ–µ –º–µ—Å—Ç–æ —Å—Ä–µ–¥–∏ —Å–æ—Å–µ–¥–Ω–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤)
        - –ø–æ–∑–∏—Ü–∏–æ–Ω–∏—Ä—É–µ—Ç—Å—è –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –±–ª–∏–∂–∞–π—à–µ–≥–æ —Ä–æ–¥–∏—Ç–µ–ª—è —Å **`position` ‚â† `static`** (–µ—Å–ª–∏ —Ç–∞–∫–æ–≥–æ –Ω–µ—Ç - –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ **`body`**)
        - —ç–ª–µ–º–µ–Ω—Ç –ø–æ —à–∏—Ä–∏–Ω–µ –∑–∞–Ω–∏–º–∞–µ—Ç —Ä–æ–≤–Ω–æ —Å—Ç–æ–ª—å–∫–æ –º–µ—Å—Ç–∞, —Å–∫–æ–ª—å–∫–æ –≤–º–µ—â–∞–µ—Ç—Å—è –∫–æ–Ω—Ç–µ–Ω—Ç–∞
    - `fixed`
        - —ç–ª–µ–º–µ–Ω—Ç –ø–æ–∑–∏—Ü–∏–æ–Ω–∏—Ä—É–µ—Ç—Å—è –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –≥—Ä–∞–Ω–∏—Ü –æ–∫–Ω–∞ –±—Ä–∞—É–∑–µ—Ä–∞
        - –≤—Å–µ–≥–¥–∞ –≤–∏–¥–µ–Ω –ø—Ä–∏ —Å–∫—Ä–æ–ª–ª–µ
        - –≤—ã—Ä—ã–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –ø–æ—Ç–æ–∫–∞
    - `sticky`
        - –≤–µ–¥—ë—Ç —Å–µ–±—è –∫–∞–∫ `relative` –¥–æ –∑–∞–¥–∞–Ω–Ω–æ–≥–æ —Å–∫—Ä–æ–ª–ª–∞, –ø–æ—Ç–æ–º "–ø—Ä–∏–ª–∏–ø–∞–µ—Ç" –∫ –ø–æ–∑–∏—Ü–∏–∏ –∏ –≤–µ–¥—ë—Ç —Å–µ–±—è –∫–∞–∫ `fixed`
        - –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –Ω—É–∂–Ω–æ –∑–∞–¥–∞–≤–∞—Ç—å —Å–≤–æ–π—Å—Ç–≤–∞ —Å–º–µ—â–µ–Ω–∏—è, —á—Ç–æ–±—ã –≤—Å—ë —Ä–∞–±–æ—Ç–∞–ª–æ
        - –∏–Ω–æ–≥–¥–∞ –º–æ–∂–µ—Ç **–Ω–µ —Å—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å** (–∏–∑-–∑–∞ —Ç–æ–≥–æ —á—Ç–æ —ç–ª–µ–º–µ–Ω—Ç —Å `position: sticky` –∑–∞–Ω–∏–º–∞–µ—Ç –≤—Å—é –≤—ã—Å–æ—Ç—É —Ä–æ–¥–∏—Ç–µ–ª—è)
            - **–§–∏–∫—Å:** –∑–∞–¥–∞—Ç—å —ç–ª–µ–º–µ–Ω—Ç—É —Å `position: sticky` –∑–Ω–∞—á–µ–Ω–∏–µ `height` –∏–ª–∏ `max-height`

# `class`

```html
<div class="card active">Несколько классов</div>
```

- Добавляет класс элементу для стилизации через CSS или работы с JS

---

# `id`

```html
<section id="contacts">Контакты</section>
<a href="#contacts">Перейти к контактам</a>
```

- Добавляет уникальный идентификатор элементу (должен быть уникален на всей странице)
- Используется для:
    - якорных ссылок (`<a href="#id">`)
    - связки `<label for="id">` с `<input id="id">`
    - работы с элементом через JS

---

# `title`

```html
<button type="button" title="Добавить в корзину">🛒</button>
<abbr title="HyperText Markup Language">HTML</abbr>
<a href="/profile" title="Перейти в профиль">👤</a>
```

- Всплывающая подсказка при наведении курсора ✅ a11y
- **хороший тон** - добавлять всем интерактивным элементам без текста (иконки, кнопки с символами)
	- полезно для пользователей, особенно для иконок без подписей

# `tabindex`

```html
<!-- фокусируемый блок (порядок по умолчанию) -->
<div tabindex="0">Блок можно выделить через Tab</div>

<!-- приоритетный порядок -->
<input type="text" tabindex="1">
<button type="button" tabindex="2">Кнопка</button>

<!-- исключить из Tab-навигации -->
<button type="button" tabindex="-1">Скрыто от Tab</button>
```

- Задаёт порядок фокусировки при навигации клавишей Tab ✅ a11y
- Делает любой элемент фокусируемым
- **Значения:**
    - `0` - элемент фокусируется в естественном порядке (браузер определяет сам)
    - `1, 2, 3...` - приоритетный порядок (числа должны идти логически)
    - `-1` - исключить из Tab-навигации
- 💡 **Лучшая практика:** использовать только `0` и не задавать числовой порядок вручную (отдавать на откуп браузеру)

# `lang`

```html
<html lang="ru">
  <body>
    <p>Это русский текст.</p>
    <blockquote lang="en">This is an English quote.</blockquote>
  </body>
</html>
```

- Указывает язык содержимого ✅ a11y, SEO
- Помогает браузеру:
	- правильно подобрать шрифт, настроить пунктуацию, подключить переводчик, озвучить текст для скринридеров
- Обычно указывается один раз в `<html lang="ru">`, но для иностранного текста внутри страницы тоже полезно

# `data-*`

```html
<input type="tel" data-mask="phone" data-region="ru">
<button type="button" data-product-id="12345" data-category="electronics">
  Купить
</button>
<div data-state="active" data-count></div>
```

```js
// чтение в JS
const input = document.querySelector('input');
console.log(input.dataset.mask);   // "phone"
console.log(input.dataset.region); // "ru"
```

- Пользовательские атрибуты для хранения данных
- Должны начинаться с `data-`
	- Значение может быть любым или отсутствовать вовсе
- Используются для:
    - передачи данных в JS через `dataset`
    - настройки библиотек и плагинов
    - хранения состояния элемента

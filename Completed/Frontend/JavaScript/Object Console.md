# Console

**Console** - встроенный объект для вывода информации в консоль браузера. Используется для отладки.

---

# Методы

## `console.log()`

```js
console.log('Внешний файл подключен');
console.log(1, 2, 3, "hello", true, d, {x: 10});
```

- выводит данные в консоль браузера
	- используется для отладки
- `console.log` принимает **произвольное количество аргументов**, необязательно 1 или 2

---

## `console.group()` и `console.groupEnd()`

```js
console.group('Группа логов');
console.log('1');
console.log('2');
console.log('3');
console.groupEnd();
```

- визуально группирует выводы в консоли
- `console.group()` - начало группы
- `console.groupEnd()` - конец группы

---

## `console.error()`

```js
console.error('это ошибка');
```

- сообщение об ошибке (красная иконка в консоли)

---

## `console.warn()`

```js
console.warn('это предупреждение');
```

- сообщение предупреждения (желтая иконка в консоли)

---

## `console.table()`

```js
console.table({ name: "Alexsey", age: 13 });
```

- создание таблицы из объекта или массива

---

## Стилизация консоли

```js
const styles = "background-color: blue; color: black; padding: 20px"; // переменная со стилями

console.log("%cStart coding, now!", styles);
```

- `%c` - означает что вторым параметром мы добавим стили
	- помимо `%c` есть еще множество других спецификаторов
- редко пригодится, но прикольная возможность

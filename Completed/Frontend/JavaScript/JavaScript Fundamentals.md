# ECMAScript

**ECMAScript** - стандарт, который определяет язык программирования JavaScript (JS). Стандарт разрабатывается организацией Ecma International и определяет синтаксис, типы данных, операторы, объекты и другие элементы, необходимые для написания программ на JS.

> **JavaScript** - стандартизированная версия реализации стандарта ECMAScript

# JIT (Just-In-Time) компиляция

**JavaScript** - динамически интерпретируемый язык с применением компиляции Just-In-Time (JIT)

**Как работает:**

- **Интерпретация:** в основном JavaScript-код интерпретируется, т.е. выполняется построчно непосредственно браузером во время выполнения
- **JIT-компиляция:** для улучшения производительности, JIT-компилятор переводит часто используемые части JavaScript-кода в машинный код для более быстрой компиляции

> В итоге JavaScript можно назвать **смешанным языком**, так как он сочетает в себе элементы интерпретации и компиляции

# Точка с запятой

```js
let userName = 'Mark'  // можно без точки с запятой
let userName = 'Mark'; // можно с точкой с запятой
```

- точку с запятой ставить **не обязательно** в конце строк кода
- но иногда могут быть ошибки

**Пример ошибки:**

```js
console.log('text')
[1, 2, 3].forEach(console.log)
// ❌ ОШИБКА! JavaScript попытается выполнить:
// console.log('text')[1, 2, 3].forEach(console.log)
```

**Правильно:**

```js
console.log('text');
[1, 2, 3].forEach(console.log);
// ✅ Работает
```

> **Рекомендация:** лучше ставить точку с запятой чтобы избежать ошибок

# Парадигмы программирования

**JavaScript** - мультипарадигменный язык программирования

Поддерживает:

- **Объектно-ориентированный стиль (OOP)**
- **Императивный стиль**
- **Функциональный стиль (FP)**

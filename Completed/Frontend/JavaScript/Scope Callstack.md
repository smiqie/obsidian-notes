# Scope - область видимости

- Определяет, где в коде доступна переменная.

---

## Глобальная область видимости (Global Scope)

Переменная объявлена вне любых блоков - доступна везде

```js
const num = 10; // global
```

---

## Локальная область видимости (Local Scope)

Переменная объявлена внутри `{}` - доступна только внутри этого блока

```js
{
    const num2 = 20; // local — снаружи недоступна
}
```

- Локальный скоуп видит глобальный, но не наоборот

```js
const num1 = 10; // global

{
    const num2 = num1; // ✅ локальный видит глобальный
}

function showScopeExample() { // тоже создаёт локальный скоуп
    const num0 = 10;
    return num1 + num0; // ✅ видит num1 из глобального скоупа
}
```

# Call Stack - стек вызовов

- Структура данных, которая отслеживает какие функции сейчас выполняются
- Работает по принципу LIFO (Last In, First Out)

- По умолчанию в Call Stack находится только `anonymous` - глобальная область видимости

## Простой вызов

```js
const log = console.log;

function one()   { log("One"); }
function two()   { log("Two"); }
function three() { log("Three"); }

one();   // one попала в стек → отработала → стек очистился
two();   // то же самое
three(); // то же самое
```

## Вложенные вызовы

- **Могут спросить на собеседованиях!**

```js
const log = console.log;

function one() {
    log("One");
    two();
}
function two() {
    log("Two");
    three();
}
function three() {
    log("Three");
}

one();
```

Состояние стека в момент выполнения `three`:

```
three      ← верх стека (выполняется сейчас)
two
one
anonymous  ← низ стека (глобальный контекст)
```

- Когда `three` завершится - она удаляется из стека, затем `two`, затем `one`
	- Стек возвращается к `anonymous`

# IIFE

- **IIFE** (Immediately Invoked Function Expression) - немедленно вызываемое функциональное выражение

- Раньше использовалось чтобы ограничить область видимости `var`, у которого нет блочного скоупа
	- Сейчас не используется - актуально только для легаси-кода

```js
(function() {
    var message = "Hello user!";
    console.log(message);
})();
console.log(message); // ❌ ReferenceError: message is not defined

// ========== Пример с аргументами: ==========
(function(name) {
    var message = "Hello!";
    console.log(message, name);
})("Anna");
```

```js
( function(name) { ... } )     ("Anna");
^^^^^^^^^^^^^^^^^^^^^^^^^^      ^^^^^^^^^
     1. ОПРЕДЕЛЕНИЕ            2. ВЫЗОВ
   (создаем функцию)      (передаем аргументы)
```

- **Суть IIFE:** Создать временную «коробку» (область видимости), выполнить в ней код и сразу её уничтожить, чтобы не мусорить в глобальной программе

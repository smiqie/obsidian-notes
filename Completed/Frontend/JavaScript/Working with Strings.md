# Working with Strings

**String** - тип данных для работы с текстом

---

# Основы

## Создание строк

```js
'text'                  // одинарные кавычки
"text"                  // двойные кавычки
`text ${variable}`      // обратные кавычки (backticks)
```

---

## Интерполяция

```js
const name = 'Max';
const a = 5, b = 3;

const msg = `name: ${name}, sum: ${a + b}`; // ✅ читабельно
```

- шаблонные строки (обратные кавычки) позволяют вставлять переменные через `${}`

---

## Конкатенация

```js
const sum = "sum: " + (a + b); // ❌ плохой тон
```

- конкатенация через `+` - устаревший способ
	- лучше использовать интерполяцию

---

## Форматирование

```js
let str = `User:
- Alex
- Max
- Julia
`;
```

- обратные кавычки сохраняют форматирование (переносы строк, отступы)

---

## Escape-последовательности

```js
"hi \nworld"  // \n - перенос на новую строку
"text \"hi\"" // \" - экранирование двойной кавычки
"path\\file"  // \\ - экранирование обратного слэша
"tab\there"   // \t - табуляция
"\u00A9"      // \uXXXX - unicode символ (©)
```

**Популярные:**

- `\n` - перенос на новую строку
- `\"` - двойная кавычка
- `\'` - одинарная кавычка
- `\\` - обратный слэш
- `\t` - табуляция
- `\uXXXX` - unicode символ (4 шестнадцатеричных символа)

---

## Unicode

```js
// Короткие коды (до 4 символов)
const copyRight = "\u00A9"; // ©
const euro = "\u20AC";      // €

// Длинные коды (эмодзи) - нужны фигурные скобки
const thumbsUp = "\u{1F44D}"; // 👍
const fire = "\u{1F525}";     // 🔥
```

- `\uXXXX` - для коротких кодов (до 4 символов)
- `\u{XXXXX}` - для длинных кодов (эмодзи)
- **фигурные скобки обязательны** для кодов длиннее 4 символов

# Свойства

## `.length` - длина строки

```js
const text = "World";

text.length; // 5
```

- **Возвращает:** число (количество символов в строке)

---

# Доступ к символам

## По индексу `[n]`

```js
const text = "World";

text[0]; // 'W'
text[4]; // 'd'
```

- доступ к символу по индексу (от 0)
- **Возвращает:** строку (один символ) или `undefined` (если индекс не существует)

---

## `.at(index)` - получить символ

**Описание:** возвращает символ по индексу (поддерживает отрицательные индексы)

**Синтаксис:**

```js
str.at(index)
```

**Параметры:**

- `index` - индекс символа (может быть отрицательным)

**Возвращает:** строку (один символ) или `undefined` (если индекс не существует)

**Пример:**

```js
const text = "World";

text.at(0);  // 'W'
text.at(-1); // 'd' (с конца)
text.at(-2); // 'l'
```

> Более современный метод чем `.charAt()`

---

## `.charAt(index)` - получить символ

**Описание:** возвращает символ по индексу

**Синтаксис:**

```js
str.charAt(index)
```

**Параметры:**

- `index` - целое число от 0 до длины строки минус 1

**Возвращает:** строку (один символ) или пустую строку `""` (если индекс не существует)

**Пример:**

```js
const text = "World";

text.charAt(0); // 'W'
text.charAt(4); // 'd'
```

> НЕ поддерживает отрицательные индексы

# Методы изменения регистра

## `.toUpperCase()` - к верхнему регистру

**Описание:** преобразует все буквы строки к верхнему регистру

**Синтаксис:**

```js
str.toUpperCase()
```

**Возвращает:** новую строку (все буквы в верхнем регистре)

**Пример:**

```js
const text = "World";

text.toUpperCase(); // 'WORLD'
```

## `.toLowerCase()` - к нижнему регистру

**Описание:** преобразует все буквы строки к нижнему регистру

**Синтаксис:**

```js
str.toLowerCase()
```

**Возвращает:** новую строку (все буквы в нижнем регистре)

**Пример:**

```js
const text = "World";

text.toLowerCase(); // 'world'
```

# Методы работы с пробелами

## `.trim()` - убрать пробелы с обеих сторон

**Описание:** удаляет пробелы с начала и конца строки

**Синтаксис:**

```js
str.trim()
```

**Возвращает:** новую строку (без пробелов с начала и конца)

**Пример:**

```js
const str = "   TEXT   ";

str.trim(); // 'TEXT'
```

> Старый метод, но чаще всего используется в разработке

## `.trimStart()` - убрать пробелы слева

**Описание:** удаляет пробелы с начала строки

**Синтаксис:**

```js
str.trimStart()
```

**Возвращает:** новую строку (без пробелов с начала)

**Пример:**

```js
const str = "   TEXT   ";

str.trimStart(); // 'TEXT   '
```

## `.trimEnd()` - убрать пробелы справа

**Описание:** удаляет пробелы с конца строки

**Синтаксис:**

```js
str.trimEnd()
```

**Возвращает:** новую строку (без пробелов с конца)

**Пример:**

```js
const str = "   TEXT   ";

str.trimEnd(); // '   TEXT'
```

# Методы повторения и дополнения

## `.repeat(count)` - повторить строку

**Описание:** повторяет строку указанное количество раз

**Синтаксис:**

```js
str.repeat(count)
```

**Параметры:**

- `count` - количество повторений

**Возвращает:** новую строку (повторенную n раз)

**Пример:**

```js
const text = "World";

text.repeat(3); // 'WorldWorldWorld'
```

---

## `.padStart(length, fillString)` - дополнить слева

**Описание:** дополняет строку слева до указанной длины

**Синтаксис:**

```js
str.padStart(length, fillString)
```

**Параметры:**

- `length` - итоговая длина строки
- `fillString` - символ для заполнения

**Возвращает:** новую строку (дополненную слева)

**Пример:**

```js
const str = "abc";

str.padStart(10, "0"); // '0000000abc'
str.padStart(5, "x");  // 'xxabc'
```

## `.padEnd(length, fillString)` - дополнить справа

**Описание:** дополняет строку справа до указанной длины

**Синтаксис:**

```js
str.padEnd(length, fillString)
```

**Параметры:**

- `length` - итоговая длина строки
- `fillString` - символ для заполнения

**Возвращает:** новую строку (дополненную справа)

**Пример:**

```js
const str = "abc";

str.padEnd(10, "0"); // 'abc0000000'
str.padEnd(5, "x");  // 'abcxx'
```

# Методы замены

## `.replace(searchValue, replaceValue)` - заменить

**Описание:** заменяет первое вхождение подстроки

**Синтаксис:**

```js
str.replace(searchValue, replaceValue)
```

**Параметры:**

- `searchValue` - что ищем
- `replaceValue` - на что заменяем

**Возвращает:** новую строку (с заменой первого вхождения)

**Пример:**

```js
const str = "JavaScript is awesome";

str.replace("Java", "Type"); // 'TypeScript is awesome'
```

> Заменяет только **первое** вхождение

# Методы поиска

## `.indexOf(searchString)` - индекс первого вхождения

**Описание:** возвращает индекс первого вхождения подстроки или -1 если не найдено

**Синтаксис:**

```js
str.indexOf(searchString)
```

**Параметры:**

- `searchString` - что ищем

**Возвращает:** число (индекс первого вхождения) или `-1` (если не найдено)

**Пример:**

```js
const str = "JavaScript is awesome";

str.indexOf("Script"); // 4
str.indexOf("Python"); // -1 (не найдено)
```

**Правильная работа с indexOf:**

```js
const str = "JavaScript is awesome";

// ❌ Неправильно
if (str.indexOf("Java")) { // 0 - falsy value
    console.log('Найдено');
}

// ✅ Правильно
if (str.indexOf("Java") !== -1) { // всегда сравнивать с -1
    console.log('Найдено');
}
```

---

## `.includes(searchString)` - проверка вхождения

**Описание:** проверяет наличие подстроки в строке

**Синтаксис:**

```js
str.includes(searchString)
```

**Параметры:**

- `searchString` - что ищем

**Возвращает:** `true` (если найдено) или `false` (если не найдено)

**Пример:**

```js
const str = "JavaScript is awesome";

str.includes("is");     // true
str.includes("Python"); // false
```

> Используется часто

**Описание:** проверяет заканчивается ли строка на указанную подстроку

**Синтаксис:**

```js
str.endsWith(searchString)
```

**Параметры:**

- `searchString` - на что должна заканчиваться строка

**Возвращает:** `true` (если заканчивается) или `false` (если нет)

**Пример:**

```js

str.endsWith("Java");    // false
```

# Методы извлечения подстроки

## `.substring(start, end)` - извлечь подстроку

**Описание:** извлекает часть строки между двумя индексами

**Синтаксис:**

```js
str.substring(start, end)
```

**Параметры:**

- `start` - индекс начала
- `end` - индекс конца (не включительно, необязательный)

**Возвращает:** новую строку (подстроку)

**Пример:**

```js
const str = "hardcore";

str.substring(0, 4); // 'hard' (с 0 до 4, не включая 4)
str.substring(4);    // 'core' (с 4 до конца)
str.substring(8, 4); // 'core' (меняет местами если start > end)
```

> Если `start > end` - меняет их местами

---

## `.slice(start, end)` - извлечь подстроку

**Описание:** извлекает часть строки между двумя индексами

**Синтаксис:**

```js
str.slice(start, end)
```

**Параметры:**

- `start` - индекс начала (может быть отрицательным)
- `end` - индекс конца (не включительно, необязательный)

**Возвращает:** новую строку (подстроку)

**Пример:**

```js
const str = "hardcore";

str.slice(0, 4); // 'hard'
str.slice(4);    // 'core'
str.slice(-4);   // 'core' (последние 4 символа)
str.slice(8, 4); // '' (пустая строка, НЕ меняет местами)
```

> Поддерживает отрицательные индексы, НЕ меняет местами start/end

## Разница между substring и slice

| Метод       | Отрицательные индексы | Если start > end         |     |
| ----------- | --------------------- | ------------------------ | --- |
| `substring` | ❌ не поддерживает     | меняет местами           |     |
| `slice`     | ✅ поддерживает        | возвращает пустую строку |     |
|             |                       |                          |     |

> НЕ рекомендуется использовать `.substr()` - считается устаревшим

# Методы объекта String

## `String.fromCodePoint()` - символ из кода

**Описание:** возвращает строку, созданную из указанного кода Unicode

**Синтаксис:**

```js
String.fromCodePoint(num1, ..., numN)
```

**Параметры:**

- `num1, ..., numN` - десятичные коды Unicode

**Возвращает:** строку (символ из кода Unicode)

**Пример:**

```js
String.fromCodePoint(128077); // 👍
String.fromCodePoint(128293); // 💥
```

**Разница способов вставки эмодзи:**
- `"🥵"` - вставка напрямую
- `"\u{1F44D}"` - шестнадцатеричный код (0x1F44D) - **самый оптимальный вариант**
- `String.fromCodePoint(128077)` - десятичный код (128077 = 0x1F44D)

---

## `String.raw()` - сохранение пути к файлу

**Описание:** возвращает строку без обработки escape-последовательностей

**Синтаксис:**

```js
String.raw`templateString`
```

**Возвращает:** строку (без обработки escape-последовательностей)

**Пример:**

```js
const filePath = String.raw`C:\Development\profile\aboutme.html`;
```

> Используется для сохранения путей к файлам (не обрабатывает `\`)

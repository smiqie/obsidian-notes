# Object Date

**Date** - встроенный объект для работы с датами и временем. Используется почти в каждом проекте.

**Важно:** месяцы начинаются с **0** (0 = январь, 11 = декабрь), дни недели тоже с **0** (0 = воскресенье)

**Важно:** создавая экземпляр класса, мы имеем больше возможностей

---

# Создание даты

## `new Date()` - текущая дата

```js
const d = new Date();
// Sun Mar 15 2026 10:30:00 GMT+0300
```

> **Возвращает:** объект Date с текущей датой, временем и часовым поясом

## `new Date(milliseconds)` - дата от timestamp

```js
const d = new Date(0);
// Thu Jan 01 1970 03:00:00 GMT+0300

const d = new Date(1000000000000);
// Sat Sep 09 2001 04:46:40 GMT+0300
```

> **Параметры:**
> 
> - `milliseconds` - миллисекунды с 1 января 1970 года (timestamp)
> 
> **Возвращает:** объект Date

---

## `new Date(year, month, ...)` - конкретная дата

```js
const d = new Date(2026, 10);              // ноябрь 2026
const d = new Date(2026, 0, 15);           // 15 января 2026
const d = new Date(2026, 0, 15, 9, 30, 33); // 15 января 2026, 09:30:33

// Переполнение месяцев
// Месяц 12 = январь следующего года, т.к. месяцы с 0 (0-11 = 2026, 12 = переход на 2027)
const d = new Date(2026, 12, 15); // 15 января 2027
```

> **Параметры:**
> 
> - `year` - год
> - `month` - месяц (от 0 до 11, где 0 = январь)
> - `day` - день (необязательный)
> - `hours` - часы (необязательный)
> - `minutes` - минуты (необязательный)
> - `seconds` - секунды (необязательный)
> 
> **Возвращает:** объект Date
> 
> **Важно:** месяцы с 0! (0 = январь, 11 = декабрь)

---

## `new Date(dateString)` - из строки

```js
// работает, но не стандарт ISO
const d = new Date("2026-08-15 09:30");

// стандарт ISO (буква Т разделяет дату и время)
const d = new Date("2026-08-15T09:30:00");

const d = new Date("08-15-2026");

// ПРИМЕР ОШИБКИ (Месяца 00 не существует в строках):
const wrong = new Date("2008-00-29");

// ПРАВИЛЬНО (в строке писать 01 вместо 00):
const trickyStr = new Date("2008-01-29");
```

- **ВАЖНО: В строковом формате месяцы как у людей (01-12), а не с нуля!**

> **Параметры:**
> - `dateString` - строка с датой
>
> **Возвращает:** объект Date

# Методы Date

## `Date.now()` - текущий timestamp

```js
Date.now(); // 1710320400000
```

> **Возвращает:** число (timestamp - миллисекунды с 1 января 1970)

---

## `Date.parse(dateString)` - строка в timestamp

```js
Date.parse("2024-08-14T00:00:00"); // 1723582800000
```

> **Параметры:**
> 
> - `dateString` - строка с датой
> 
> **Возвращает:** число (timestamp)
> 
> Используется для конвертации строки даты (например, от бэкенда) в timestamp

---

# Геттеры (получение значений)

## `.getFullYear()` - получить год

```js
const d = new Date();

d.getFullYear(); // 2026
```

> **Возвращает:** число (год)


## `.getMonth()` - получить месяц

```js
const d = new Date("2026-08-15");

d.getMonth();     // 7 (август, отсчет с 0)
d.getMonth() + 1; // 8 (август, логический номер месяца) 
```

> **Возвращает:** число (месяц от 0 до 11)
> 
> **Важно:** месяцы с 0! Для логического номера добавить `+ 1`


## `.getDate()` - получить день месяца

```js
const d = new Date("2026-08-15");

d.getDate(); // 15
d.getDate(-1); // последнее число месяца
```

> **Возвращает:** число (день месяца от 1 до 31)


## `.getDay()` - получить день недели

```js
const d = new Date("2026-03-13"); // пятница

d.getDay(); // 5 (пятница)
```

> **Возвращает:** число (день недели от 0 до 6, где 0 = воскресенье)


## `.getHours()` - получить часы

```js
const d = new Date("2026-08-15T09:30:00");

d.getHours(); // 9
```

> **Возвращает:** число (часы от 0 до 23)


## `.getMinutes()` - получить минуты

```js
const d = new Date("2026-08-15T09:30:00");

d.getMinutes(); // 30
```

> **Возвращает:** число (минуты от 0 до 59)


## `.getSeconds()` - получить секунды

```js
const d = new Date("2026-08-15T09:30:45");

d.getSeconds(); // 45
```

> **Возвращает:** число (секунды от 0 до 59)


## `.getTime()` - получить timestamp

```js
const d = new Date();

d.getTime(); // 1710320400000
```

> **Возвращает:** число (timestamp - миллисекунды с 1 января 1970)


## `.valueOf()` - получить timestamp

```js
const d = new Date();

d.valueOf(); // 1710320400000
```

> **Возвращает:** число (timestamp)
> 
> Аналог `.getTime()`

# Сеттеры (установка значений)

> **Важно:** все сеттеры сразу изменяют объект Date!

## `.setFullYear(year, month, day)` - установить год

```js
const d = new Date();

d.setFullYear(2025);        // установить год
d.setFullYear(2025, 0, 10); // установить год, месяц и день
```

> **Параметры:**
> 
> - `year` - год
> - `month` - месяц (необязательный)
> - `day` - день (необязательный)
> 
> **Возвращает:** число (timestamp)


## `.setMonth(month, day)` - установить месяц

```js
const d = new Date();

d.setMonth(0);    // январь
d.setMonth(11);   // декабрь
d.setMonth(0, 15); // 15 января
```

> **Параметры:**
> 
> - `month` - месяц (от 0 до 11)
> - `day` - день (необязательный)
> 
> **Возвращает:** число (timestamp)


## `.setDate(day)` - установить день месяца

```js
const d = new Date("2026-08-15");

d.setDate(1);  // 1 августа
d.setDate(31); // 31 августа
d.setDate(0);  // последний день предыдущего месяца (31 июля)
d.setDate(-1); // предпоследний день предыдущего месяца (30 июля)
```

> **Параметры:**
> 
> - `day` - день месяца
> 
> **Возвращает:** число (timestamp)
> 
> **Особенность:**
> 
> - `0` - последний день предыдущего месяца
> - `-1` - предпоследний день предыдущего месяца
> - **и т.д.**


## `.setHours(hours)` - установить часы

```js
const d = new Date();

d.setHours(0);  // 00:00
d.setHours(15); // 15:00
```

> **Параметры:**
> 
> - `hours` - часы (от 0 до 23)
> 	- Если передать число больше 23, JS не выдаст ошибку, а **перекинет дату на следующий день**:
> 		- `d.setHours(24)` - установит 00:00 **следующего** дня
> 		- `d.setHours(48)` - установит 00:00 через **два** дня
> 		- `d.setHours(-1)` - установит 23:00 **предыдущего** дня (откат назад)
> 
> **Возвращает:** число (timestamp)


## `.setMinutes(minutes)` - установить минуты

```js
const d = new Date();

d.setMinutes(0);  // 0 минут
d.setMinutes(30); // 30 минут
```

> **Параметры:**
> 
> - `minutes` - минуты (от 0 до 59)
> 
> **Возвращает:** число (timestamp)


## `.setSeconds(seconds)` - установить секунды

```js
const d = new Date();

d.setSeconds(0);  // 0 секунд
d.setSeconds(45); // 45 секунд
```

> **Параметры:**
> 
> - `seconds` - секунды (от 0 до 59)
> 
> **Возвращает:** число (timestamp)

# Форматирование дат

## `Intl.DateTimeFormat()` - создать форматтер

```js
const date = new Date();

// Создаем форматтер (JavaScript автоматически создает объект, можно и без new)
const formatter = Intl.DateTimeFormat("ru-RU");
// Аналогично: new Intl.DateTimeFormat("ru-RU")

// Применяем форматтер к дате
formatter.format(date); // "15.03.2026"
```

> **Параметры .DateTimeFormat():**
> 
> - `locale` - локаль (например: `"ru-RU"`, `"en-GB"`, `"default"`)
> - `options` - объект с опциями (необязательный)
> 
> **Возвращает:** объект-форматтер
> 
> `Intl` - глобальный объект для интернационализации

**Примеры с разными локалями:**

```js
const date = new Date();

Intl.DateTimeFormat("default").format(date); // "3/15/2026"
Intl.DateTimeFormat("en-GB").format(date);   // "15/03/2026"
Intl.DateTimeFormat("ru-RU").format(date);   // "15.03.2026"
```

**С опциями:**

```js
Intl.DateTimeFormat("ru-RU", {
    day: "numeric",
    month: "long",
    year: "numeric"
}).format(date);
// "15 марта 2026 г."
```

> https://metanit.com/web/javascript/18.2.php список возможных значений
> ==`dateStyle` & `timeStyle`==

## `.format(date)` - применить форматирование

```js
const formatter = Intl.DateTimeFormat("ru-RU");

formatter.format(new Date()); // "15.03.2026"
```

> **Параметры:**
> 
> - `date` - объект Date
> 
> **Возвращает:** строку (отформатированную дату)
> 
> Применяет форматирование к дате. Обычно вызывается сразу после создания форматтера: `Intl.DateTimeFormat("ru-RU").format(d)`

---

## `.toLocaleString()` - альтернативное форматирование

```js
const d = new Date();

d.toLocaleString("ru-RU", {
    day: "numeric",
    month: "long",
    year: "numeric",
    hour: "numeric",
    minute: "2-digit",
    timeZone: "Europe/Moscow"
});
// "15 марта 2026 г., 14:57"
```

> **Параметры:**
> 
> - 1-й - локаль
> - 2-й - опции (порядок ключей не важен)
> 
> **Возвращает:** строку (отформатированную дату)
> 
> Использует `Intl` под капотом, более затратный по производительности, но разница незаметна

**Опции:**

- `day`, `month`, `year`, `hour`, `minute`, `second`
- `timeZone` - часовой пояс (например: `"Europe/Moscow"`, `"Europe/Berlin"`, `"Australia/Sydney"`)

# UTC vs ISO

## `.toUTCString()` - UTC формат

```js
const now = new Date();

now.toUTCString();
// "Tue, 20 Aug 2024 14:57:01 GMT"
```

> **Возвращает:** строку в UTC формате (читаемый вид)
> 
> **UTC (Coordinated Universal Time)** - стандарт времени используемый по всему миру
> 
> Используется для отображения даты и времени в текстовом виде (когда важна читаемость)


## `.toISOString()` - ISO формат

```js
const now = new Date();

now.toISOString();
// "2024-08-20T14:57:01.706Z"
```

> **Возвращает:** строку в ISO 8601 формате
> 
> **ISO (International Organization for Standardization)** - международный стандарт для представления дат и времени
> 
> Используется для хранения данных и передачи между системами (формат строгий и однозначный)


**Когда использовать:**

- `.toUTCString()` - для отображения (читаемость)
- `.toISOString()` - для передачи на бэкенд/хранения (строгий формат)

# Практические примеры

## Разница между датами в днях

```js
const now = new Date();
const past = new Date("2024-08-20T00:00:00");

// Получаем разницу в миллисекундах
const diffMs = now.getTime() - past.getTime();

// Преобразуем в дни
const diffDays = diffMs / 1000 / 60 / 60 / 24 / 365.25;
//                        сек   мин  час  дни   года

console.log(diffDays);
```

> Используется для таймеров, проверки подписок, показа рекламы
---

## Получить последний день месяца

```js
const d = new Date(2026, 8, 0); // 0 = последний день предыдущего месяца
// 31 августа 2026

const d2 = new Date(2026, 1, 0); // → Последний день Января (31-е)

// Правило: Чтобы найти последний день нужного месяца, нужно написать индекс следующего месяца и поставить число 0
```

# Логические операторы

## `||` - ИЛИ

Возвращает **первое истинное (truthy) значение** или последнее значение:

```js
const a = '' || null || false || 0 || 'Hi';
console.log(a); // 'Hi' (первое truthy значение)

const b = '' || null || false || 0;
console.log(b); // 0 (последнее значение, если нет truthy)
```

> Вычисляет операнды слева направо
> 
> Каждый операнд преобразуется в логическое значение
> 
> Если результат `true` - останавливается и возвращает исходное значение этого операнда
> 
> Если все операнды ложны - возвращается последний

**Примеры:**

```js
false || true           // true
0 || 100               // 100
'' || 'Hello'          // 'Hello'
null || undefined || 5 // 5
```

> **Важно:** не писать много операторов подряд - код становится нечитаемым

## `&&` - И

Возвращает **первое ложное (falsy) значение** или последний операнд (если все истинные):

```js
const a = 'Hi' && true && 5 && null;
console.log(a); // null (первое falsy значение)

const b = 1 && 2 && 3;
console.log(b); // 3 (все truthy, возвращает последний)

const c = 10 && 0 && 20 && "Hello";
console.log(c); // 0 (первое falsy значение)
```

> Вычисляет операнды слева направо
> 
> Каждый операнд преобразуется в логическое значение
> 
> Если результат `false` - останавливается и возвращает исходное значение этого операнда
> 
> Если все операнды истинны - возвращается последний

**Разница между `&&` и `||`:**

- `&&` возвращает первое **ложное** значение
- `||` возвращает первое **истинное** значение

**В React часто встречается:**

```js
{isLoading && <div>Loading...</div>}
// Если isLoading = true, отрисуется div
```

## ❌ Частая, но не рекомендуемая практика

```js
const age = 18;
const isYoung = age < 18;

// ❌ Плохо / если isYoung - true, выполнится инструкция после &&
isYoung && console.log('Close');

// ✅ Хорошо - используй if
if (isYoung) {
    console.log('Close');
}
```

## Приоритет операторов

`&&` имеет **приоритет выше**, чем `||`:

```js
true || false && false
// интерпретируется как:
true || (false && false)
// результат: true
```

## `!` - НЕ (логическое отрицание)

Приводит аргумент к логическому типу и возвращает противоположное значение:

```js
!true     // false
!false    // true
!0        // true
!1        // false
!'text'   // false
!''       // true
```

**Как работает:**

```js
let c = !0;
// 1) 0 → false
// 2) !false → true
// Результат: true
```


## `!!` - двойное отрицание

Приводит аргумент к логическому типу (без реверса):

```js
!!0       // false
!!1       // true
!!'text'  // true
!!''      // false

!!0 === Boolean(0)  // true
```

> **Важно:** `!!` = `Boolean()`
> 
> **✅ Рекомендуется:** использовать `Boolean()` - читабельнее

# Тернарный оператор

```js
const age = 18;
const status = age >= 18 ? 'Adult' : 'Minor';
```

> **Синтаксис:** `условие ? значение_если_true : значение_если_false`
> 	`?` = if
> 	`:` = else

**Вложенный тернарный оператор:**

```js
const status = age >= 18
    ? 'Adult'
    : age < 18
        ? 'Child'
        : 'Nothing';
```

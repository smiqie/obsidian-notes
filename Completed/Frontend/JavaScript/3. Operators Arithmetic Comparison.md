# `typeof` - определение типа

```js
const name = 'Max';
console.log(typeof name); // "string"
```

> **Возвращает:** строку с типом переменной

# Арифметические операторы

```js
+   // сложение
-   // вычитание
*   // умножение
/   // деление
%   // остаток от деления
**  // возведение в степень
```

**Возведение в степень:**

```js
const x = 2 ** 3; // 8 (2 в степени 3)
```

# Инкремент и декремент

## Постфиксный (`a++` / `a--`)

```js
let a = 10;
let b = a++;
// a = 11
// b = 10 (сначала присваивается, потом увеличивается)
```

> Сначала возвращает значение, **потом** увеличивает/уменьшает


## Префиксный (`++a` / `--a`)

```js
let a = 10;
let b = ++a;
// a = 11
// b = 11 (сначала увеличивается, потом присваивается)
```

> Сначала увеличивает/уменьшает, **потом** возвращает значение

# Операторы сравнения

```js
==   // равно (с приведением типов) ❌ устаревший
===  // строгое равно ✅ рекомендуется
!=   // не равно (с приведением типов) ❌ устаревший
!==  // строго не равно ✅ рекомендуется
>    // больше
<    // меньше
>=   // больше или равно
<=   // меньше или равно
```


## Строгое сравнение (`===` / `!==`)

```js
2 === '2'  // false (разные типы)
2 !== 2    // false

2 == '2'   // true ❌ (приводит к одному типу)
2 === '2'  // false ✅ (учитывает тип)

true == 1  // true ❌
true === 1 // false ✅
```

> **`==`** - сравнивает с приведением к одному типу (в большинстве случаев к number), `null` и `undefined` не приводятся к числу - они просто равны друг другу и не равны больше ничему
> 
> **`===`** - сравнивает по типу И по значению, без приведения
> 
> **✅ Рекомендуется:** всегда использовать `===` и `!==`

## Сравнение строк

```js
'JavaS' > 'Java'  // true (длина + сравнение позиционно)
'ы' > 'f'         // true (кириллица идет позже латиницы в Unicode)
```

> **Важно:** строки сравниваются по порядку символов в Unicode

# Операторы присваивания

```js
=    // присваивание
+=   // сложение и присваивание
-=   // вычитание и присваивание
*=   // умножение и присваивание
/=   // деление и присваивание
%=   // остаток и присваивание
**=  // возведение в степень и присваивание
```

**Примеры:**

```js
d = d + 1;  // аналогично: d += 1;
d = d * 2;  // аналогично: d *= 2;
```

**Для строк `+=` - конкатенация:**

```js
let str = "Hello";
str += " world"; // "Hello world"
```

# Классификация операторов

## Унарный оператор

Работает с **одним** операндом:
	`-x`, `+x`, `!x`, `typeof x`, `++x`, `x--`

## Бинарный оператор

Работает с **двумя** операндами:
	`a + b`, `a - b`, `a > b`, `a && b`

## Тернарный оператор

Работает с **тремя** операндами:
	`условие ? выражение1 : выражение2`


---

# Унарный плюс и минус

## Унарный минус (`-`)

Делает число отрицательным:

```js
const a = 10;
console.log(-a); // -10
```

## Унарный плюс (`+`)

Преобразует значение к числу:

```js
+'123'   // 123
+'3.14'  // 3.14
+true    // 1
+false   // 0
+'abc'   // NaN
```

> Аналогично `Number('123')`

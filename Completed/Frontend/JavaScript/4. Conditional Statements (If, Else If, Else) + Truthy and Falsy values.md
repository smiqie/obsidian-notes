# Conditional Statements

---

# If/Else

```js
if (условие) { // 1я ветка
    // код
} else if (другое_условие) { // 2я ветка
    // код
} else { // доп ветка
    // код
}
```

- все что происходит в конструкции if, в скобках - преобразуется к логическому типу

---

# Falsy values (ложные значения)

**Значения которые преобразуются в `false`:**

```js
if (0) {}           // false
if ('') {}          // false (пустая строка)
if (NaN) {}         // false
if (null) {}        // false
if (undefined) {}   // false
```

- число `0`
- пустая строка `""`
- `NaN`
- `null`
- `undefined`

**Могут спросить на собесе!**

---

# Truthy values (правдивые значения)

**Значения которые преобразуются в `true`:**

```js
if (1) {}           // true
if (' ') {}         // true (строка с пробелом)
```

- все остальные значения

# Math Object

**Math** - встроенный глобальный объект для работы с числами. Встречается часто в разработке.

---

# Методы Math

## `Math.min()` - минимальное число

```js
Math.min(4, 7, 2, 13); // 2
Math.min(10, 5, 8);    // 5
```

**Параметры:** любое количество чисел
**Возвращает:** наименьшее из переданных чисел

## `Math.max()` - максимальное число

```js
Math.max(4, 7, 2, 13); // 13
Math.max(10, 5, 8);    // 10
```

**Параметры:** любое количество чисел
**Возвращает:** наибольшее из переданных чисел

## `Math.ceil()` - округление вверх

```js
Math.ceil(5.3);  // 6
Math.ceil(5.9);  // 6
Math.ceil(5.01); // 6
```

**Параметры:** дробное число
**Возвращает:** число округленное **в большую сторону**

## `Math.floor()` - округление вниз

```js
Math.floor(5.3);  // 5
Math.floor(5.9);  // 5
Math.floor(5.99); // 5
```

**Параметры:** дробное число
**Возвращает:** число округленное **в меньшую сторону**

## `Math.round()` - округление к ближайшему

```js
Math.round(5.3); // 5 (ближе к 5)
Math.round(5.5); // 6 (ближе к 6)
Math.round(5.9); // 6 (ближе к 6)
```

**Параметры:** дробное число
**Возвращает:** число округленное **к ближайшему целому**

## `Math.trunc()` - отбросить дробную часть

```js
Math.trunc(3.324);  // 3
Math.trunc(5.999);  // 5
Math.trunc(-3.324); // -3
```

**Параметры:** дробное число
**Возвращает:** целую часть числа (удаляет дробные знаки)

## `Math.pow()` - возведение в степень

```js
Math.pow(3, 4); // 81 (3^4 = 3 * 3 * 3 * 3)
Math.pow(2, 3); // 8  (2^3 = 2 * 2 * 2)
Math.pow(5, 2); // 25 (5^2 = 5 * 5)
```

**Параметры:**
- 1-й аргумент - число которое возводим
- 2-й аргумент - степень

**Возвращает:** число возведенное в степень

## `Math.sqrt()` - квадратный корень

```js
Math.sqrt(81);  // 9  (√81 = 9)
Math.sqrt(16);  // 4  (√16 = 4)
Math.sqrt(100); // 10 (√100 = 10)
```

**Параметры:** число
**Возвращает:** квадратный корень числа (обратная операция `Math.pow()`)

## `Math.abs()` - модуль числа

```js
Math.abs(-48);  // 48
Math.abs(48);   // 48
Math.abs(-100); // 100
```

**Параметры:** число
**Возвращает:** абсолютное значение числа (модуль)

## `Math.random()` - случайное число

```js
// случайное число от 0 (включительно) до 1 (не включительно)
Math.random(); // например: 0.123456789, 0.987654321
```

**Параметры:** нет
**Возвращает:** случайное **дробное** число от `0` до `1` (не включая `1`)

- **Генерация случайного числа в диапазоне:**

```js
// Случайное число от 1 до 50 (включая и 1 и 50)
Math.floor(Math.random() * 50 + 1);

// Разбор формулы:
// Math.random()           → 0.456 (случайное от 0 до 1)
// Math.random() * 50      → 22.8 (умножаем на 50)
// Math.random() * 50 + 1  → 23.8 (добавляем 1 чтобы начиналось с 1, а не с 0)
// Math.floor(...)         → 23   (округляем вниз)
```

# Свойства Math

## `Math.PI` - число π

```js
Math.PI; // 3.141592653589793
```

- математическая константа π (пи)

## `Math.E` - число Эйлера

```js
Math.E; // 2.718281828459045
```

- основание натурального логарифма

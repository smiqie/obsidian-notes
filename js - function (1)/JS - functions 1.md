---
- аргументы - это хначение этого параметра, т.ю.е значение которые передается при выовзове этого функции. значение при выовзве функции
- параметры это  перемнная указываемая в круглых скобках в фнукции. значение при объеклвание функции
```js
function sum(num1, num2) // parameters
{ 
    console.log(num1 + num2);
}

sum(5, 5); // arguments
```


- ключевоео слово `return` - возвращает какоето значение из функции? без return функция возвращет undefined?

---
- функции по сути это действие, поэтому название сункций обычно являюься глаголом (нпр showMessage(), getName(), calc(), crete(), check() etc)

# Call Stack (стек вызвова)

- 1й прмер
```js
const log = console.log;

function one(){
	log("One");
}
function two(){
	log("Two");
}
function three(){
	log("Three");
}

// По умолчнаию в Call Stack нахходитсяь только anonymous (ощначает глобальную область видимости)

one();
// Вызываем функцию
// Попадаем в функцию и в Call Stack попадает функция one
// one() функция отрабатывает и Call Stack очищается (ь.е. там опять anonymous)


// также же и с ними
two();
three();
```

**могут спросить на собесакх!**

- 1й прмер
```js
const log = console.log;

function one(){
	log("One");
	two();
}
function two(){
	log("Two");
	three();
}
function three(){
	log("Three");
}

// По умолчнаию в Call Stack нахходитсяь только anonymous (ощначает глобальную область видимости)

one();
// Вызываем функцию
// Попадаем в функцию + в Call Stack попадает функция one
// one() функция отрабатывает и мы вызываем функцию two
// Попадаем в функцию two + в Call Stack попадает функция two
// two() функция отрабатывает и мы вызываем функцию three
// Попадаем в функцию three + в Call Stack попадает функция three

// В стеке получается чтото такое: 
// three -- в самом верху
// two
// one
// anonymous - в самом низу

// потом когда фугнкция three отработает мы начнем вызходит из всех вызванных фунеций поочереждно (т.е. снчала из стекак удалиться самое верзхнее знаечние - three - при выходе из функции, потом two  - при выходе из функции а потом уже one  - при выходе из функции и стек полностью очистится)
```

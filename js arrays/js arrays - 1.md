- массив - стркутура данных для хранения упорядоченных коллекций (нпр список пользователей, товаров, элементов)
- создение массива
```js
// 1
const arr = [];
// 2
const arr = new Array();
```
- объеклвение и иницализация массива?
```js
const numbers = [1, 10, 21];
/*под капотом это выглдяит как ключ знаечния:
0: 1
1: 10
2: 21
length: 3
*/


// получение элементов массива
numbers[0] // 1
numbers[2] // 21

// массив может быть с различными значениями
const arrayMix = [1, 2, "hello", true, undefined, null, {}, []];

// массив который выключает в себя массив (как элемент) - называется двумерным, пример:
const arrayMix = [[]];

// копирвоание значения элеемнта в массиве в пемренную
let x = numbers[1]; // x = 10

// переприсваивание элемента в массиве (изменить значение у элемента), даже если массив был обяълен как const (т.к. мы не меняем сам массив а меняем его элементы)
numbers[1] = 100; // numbers[1] = 100



----

или создание через класс аррей
const arr = new Array(1, 2, 34, true); // резульата будет такой же
```
- перечисление элементов массив анаичнаются с нулеовго индекса!
- свойства массива:
	1. `const numbers = [1, 2, 3]`
	2. `numbers.length` - возвращает кол-во элементов в массиве

- копирование массивов
```js
const numbers = new Array(10, 20 ,30);
const copiedArray = numbers;
console.log(numbers === copiedArray); // true (тк они обеъкты)
```

- методы массивов:
	1. `const arr = new Array(1, 2, 3, 4, 5);`
	2. `arr.unshift(111)` - добавляет новый элемент в начало массива, отасльные элементы смещаются
		- `(111, 1, 2, 3, 4, 5)`
	3. `arr.push(777)` - добавляет новый элемент в конец массива, одлин из самых попялрных методов - **более предпочтительней чем метод unshift, тк чтобы положить элемент в начало массива - более затртаная операция чем в конец**
		- `(1, 2, 3, 4, 5, 777)`
	4. `arr.shift()` - удаляет элемент из начала массива и соответственно он его еще и возвращает
		-  `(2, 3, 4, 5)`
	5. `arr.pop()` - удаляет элемент из конца массива // и соответственно он его еще и возвращает?
		-  `(1, 2, 3, 4)`
	6. `arr.sort((a, b) => a - b)` - сортирвока массива по возрастанию, принимает в себя колюэк функцию ()
		1. `arr.sort((a, b) => b - a)` - сортирвока массива по убывнаию, принимает в себя колюэк функцию
		   или `arr.sort((a, b) => a - b).reverse()`
		- часто используются в реальной разработке

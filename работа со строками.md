- `let text = "World";`
	- `text.length` - свойство, возвращающее длину строки (кол-во симолов)
	- `text.toUpperCaes()` - метод преобразовывающий все буквы строки к верхнему регистру
	- `text.toLowerCase()` - метод преобразовывающий все буквы строки к нижнему регистру
	- `text.repeat(3)` - метод, повторящий строку нное количество раз
	- `text.at(3)` - метод, позволяющий получить символ строки по индексу (по сути альтернатива `text[3]`) . если передать -1 то мы будем идти с конца (вернется d) (-0 не бывает) - БОЛЕЕ СОВРЕМЕННЫЙ МЕТОД
	- `text.charAt(-1)` тоже самое что и просто метод `at` (но не получиться передавать отрицательные индексы)
	- `let str = "   TEXT   ";`
		- `str.trimStart()` - убирает все пробелы с левой стороны строки (с начала)
		- `str.trimEnd()` - убирает все пробелы с правой стороны строки (с конца)
		- `str.trim()` - убирает все пробелы с двух сторон (древний метод, но используется чаще всего в релаьной разработке). используется для проверки?
	- `let str = " JavaScript is awesome  "`
		- `str.Replace("Java", "Type")` заменяет подстроку из строки на подстроку (первый параметр - что ищем, второй если найдем 1й то заменим)
		- `str.indexOf("Script")` - возврщает индекс первого вхождения символа или подстроки . если ничсего не найдено возвращает -1
		- правильная работа с мтеодом indexOf
		  ```js
		  let str = "JavaScript is awesome";
		  
		  // непарвильно
		  if(str.indexOf("Java")){ // 0 - falsy value
		  
		  }
		  
		  //праивльно (нужно всегда сравнивать с -1)
		  	if(str.indexOf("Java") !== -1){ // (0 !== -1) - truthy value
		  }
		  ```

- `let str = "JavaScript is";`
	- `str.includes("is")` проверяет вхождение подстроки в строку и возвращает булевое значение, в данном случае тру. используется часто
	- `str.startsWith("Java")` проверяет начинается ли строка с подстроки  и возвращает булевое значение, в данном случае тру
	- `str.endsWith("Java")` проверяет заканчивается ли строка на подстроку и возвращает булевое значение, в данном случае фалсе

- `let str = "hardcore";`
	- аргументы методов
		- `substring(start, end)`
		- `slice(start, end)`
		- `substr(start, length)` - УСТАРЕВШИЙ (НЕ ИСПОЛЬЗОВАТЬ)
	- `str.substring(0, 4);` --> hard, подрезает строку и возвращает подстроку (начиная с 0го индекс и заканчивая 4м - не вклюячая его)
		- `str.substring(4);` --> core, подрезает строку и возвращает подстроку (начиная с 4го индекса и вклюбчая его и до конца строки)
		- `str.substring(8, 4);` --> core, если start больше чем end то он просто менеяет их местами // == (4, 8)
	- `str.slice(0, 4)` - hard (тоже самое)
		- `str.slice(8, 4)` ничего не поулчаем, просто пустую строку

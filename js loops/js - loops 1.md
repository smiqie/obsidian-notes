### **While**

```js
while (условие) {
    // код
}
```

### **Do While**

```js
do {
    // код
} while (условие);
```

- использовать этот цикл не рационально! обычно выбирабт цикл While
- по факту это синтаксический сахар
	-  синтаксический сахар - возможность записать логику разными способами и поулчить один и тот же резльтат / т.е. используем разный синтаксис для получени яодного и того же резулаьтата

### **For**

```js
// Шаги выполнения:
// 0. let i = 0; выполняется 1 раз

// 1. i < 5; проверка условия, выполняется в начале кажлой итарации
// 2. {// код} - выполянестя код внутри цикла (если условие truthy)
// 3.  i++ - выполняется инкремент/декремент

for (let i = 0; i < 5; i++) {
    // код
}

// Можно делать так, но нет смысла (т.к. есть while)
let i = 0;
for (; i < 4; ) {
    i++;
}

// бесконечный цикл (так никто не делает)
for(;;){
}
```

```js
// ПЕРЕБОР МАССИВА
const cars = ["audi", "mazda", "tesla"];

// с начала массива
for (let i = 0; i < cars.length; i++) {
    console.log(cars[i]);
}

// с конца массива / в обратном порядке
for (let i = cars.length - 1; i >= 0; i--) {
    console.log(cars[i]);
}
```


 
- обычно принято писать букву `i` в роли счетчика

### **Ключевые слова**

- `break` - выход из цикла
- `continue` - переход к следующей итерации
- **итерация** - повторение 1го круга цикла? как лучше сформулировать и кратко


- Пример работы:
	- у нас есть массив объектов полученный с сервера и нам нужно пробежаться по нему и выести данные элементы
```js
const users = [
	{ id: 1, name: "Alex", age: 35, position: "manager" },
	{ id: 2, name: "Kate", age: 22, position: "designer" },
	{ id: 3, name: "Nikita", age: 29, position: "developer" },
];

for (let i = 0; i < users.length; i++) {
    console.group(`Employee ${i + 1}`);
    console.log(`Name: ${users[i].name}`); // users[i] - элемент массива, который по сути объект, так что по сути мы обращаемся к элементу массива и вызываем у него свойство name 
    console.log(`Age: ${users[i].age}`);
    console.log(`Position: ${users[i].posirion}`);
    console.groupEnd();
}
```

- называется пузырьковой потмоу что наибольший элеемнт выпслыывает в конец массивв?
```js
// Пузырьковая сортировка (хорошо подходит под мелкие массивы)
const array = [4, 43, 2, 1];

for(let i = 0; i < array.length; i++){

	for(let j = 0; j < array.length - 1 - i; j ++){
		if(array[j] > array[j + 1]){
			 const temp = array[j];
			 array[j] = array[j + 1];
			 array[j + 1] = temp; 
		}
	}
}
```

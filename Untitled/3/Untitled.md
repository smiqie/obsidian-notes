

# `filter`

```css
/* без фильтров (по умолчанию) */
filter: none;

/* размытие */
filter: blur(5px);

/* яркость (0 = чёрный, 1 = норма, >1 = светлее) */
filter: brightness(0.5); /* затемнить на 50% */
filter: brightness(1.5); /* осветлить на 50% */

/* контрастность (0 = серый, 1 = норма, >1 = больше контраста) */
filter: contrast(200%);

/* оттенки серого (0 = цветной, 1 = чёрно-белый) */
filter: grayscale(100%);
filter: grayscale(0.5); /* 50% серого */

/* инверсия цветов (0 = норма, 1 = инверсия) */
filter: invert(1);

/* насыщенность (0 = обесцвечено, 1 = норма, >1 = насыщеннее) */
filter: saturate(2);

/* оттенок (поворот цветового круга) */
filter: hue-rotate(90deg);
filter: hue-rotate(180deg);

/* сепия (старое фото) (0 = норма, 1 = полная сепия) */
filter: sepia(100%);

/* прозрачность (0 = прозрачно, 1 = непрозрачно) */
filter: opacity(0.5);

/* тень (аналог box-shadow, но учитывает форму, включая прозрачные пиксели) */
filter: drop-shadow(5px 5px 10px rgb(0 0 0 / 0.5));

/* множественные фильтры (применяются по порядку) */
filter: brightness(1.2) contrast(1.1) saturate(1.3);
filter: grayscale(100%) brightness(1.2);

```

- применяет **графические эффекты** к элементу (работает с любыми элементами, включая изображения, текст, контейнеры)

- **Основные функции:**
    
    1. **`blur(radius)`** - размытие по Гауссу***
        - `radius` - радиус размытия (**обычно указывают `px`**)
        - чем больше значение - тем сильнее размытие
    2. **`brightness(amount)`** - яркость***
	    - **Значения:** в формате коэффициента  или %
        - `0` - полностью чёрный
        - `1` - оригинал (100%)
        - `>1` - светлее
    3. **`contrast(amount)`** - контрастность***
	    - **Значения:** в формате коэффициента  или %
        - `0` - полностью серый
        - `1` - оригинал (100%)
        - `>1` - больше контраста
    4. **`grayscale(amount)`** - оттенки серого (черно-белый эффект)***
        - `0` - цветное (0%) - по умолчанию
        - `1` или `100%` - чёрно-белое`
    5. **`invert(amount)`** - инверсия цветов***
        - `0` - оригинал (0%)
        - `1` или `100%` - полная инверсия
    6. **`saturate(amount)`** - насыщенность***
        - `0` - обесцвеченное
        - `1` - оригинал
        - `>1` - более насыщенное
    7. **`hue-rotate(angle)`** - поворот оттенка (сдвиг цветов на `x°`)***
        - значение в `deg` (градусах)
        - `0deg` - оригинал
        - `360deg` - полный круг (снова оригинал)
    8. **`sepia(amount)`** - эффект сепии (старое фото)***
        - `0` - оригинал (0%)
        - `1` или `100%` - полная сепия
    9. **`opacity(amount)`** - прозрачность***
    	- **Значения:** в формате коэффициента  или %
        - `0` - полностью прозрачно (0%)
        - `1` - полностью непрозрачно (100%)
        - **Интересный факт:** Работает быстрее свойства `opacity` за счёт аппаратного ускорения (но разница ничтожна)
    10. **`drop-shadow(offset-x offset-y blur-radius color)`** - тень***
        - в отличие от `box-shadow`, учитывает форму элемента, включая прозрачность
        - идеально для PNG с прозрачным фоном (**Частый приём**)
        - Паттерн: `drop-shadow(x y blur color)`

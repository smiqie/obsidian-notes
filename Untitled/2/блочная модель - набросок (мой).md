 # `box-sizing`

```css
/* –†–∞–∑–º–µ—Ä—ã –ù–ï–ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã: width/height —É—á–∏—Ç—ã–≤–∞—é—Ç —Ç–æ–ª—å–∫–æ –∫–æ–Ω—Ç–µ–Ω—Ç.
   –ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ padding –∏–ª–∏ border —ç–ª–µ–º–µ–Ω—Ç —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –±–æ–ª—å—à–µ. */
box-sizing: content-box;

/* –†–∞–∑–º–µ—Ä—ã –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã: width/height –≤–∫–ª—é—á–∞—é—Ç –∫–æ–Ω—Ç–µ–Ω—Ç + padding + border.
   –ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ –æ—Ç—Å—Ç—É–ø–æ–≤ –∏–ª–∏ –≥—Ä–∞–Ω–∏—Ü –æ–±—â–∏–π —Ä–∞–∑–º–µ—Ä –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è. */
box-sizing: border-box;
```
1. `box-sizing` –≤–ª–∏—è–µ—Ç —Ç–æ–ª—å–∫–æ –Ω–∞ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ —Ä–∞–∑–º–µ—Ä—ã —ç–ª–µ–º–µ–Ω—Ç–∞: `content`, `padding`, `border`

# `padding` + –ø–æ–¥—Å–≤–æ–π—Å—Ç–≤–∞

```css
/* –∑–∞–¥–∞—ë—Ç –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ –æ—Ç—Å—Ç—É–ø—ã - –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –º–µ–∂–¥—É –∫–æ–Ω—Ç–µ–Ω—Ç–æ–º –∏ –≥—Ä–∞–Ω–∏—Ü–∞–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞ */

/* –∑–∞–¥–∞—ë—Ç –æ—Ç—Å—Ç—É–ø—ã –æ—Ç–¥–µ–ª—å–Ω–æ –¥–ª—è –∫–∞–∂–¥–æ–π —Å—Ç–æ—Ä–æ–Ω—ã */

/* –¥–ª—è –≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω—ã—Ö –æ—Ç—Å—Ç—É–ø–æ–≤ (–≤–ª–µ–≤–æ/–≤–ø—Ä–∞–≤–æ) */

/* –¥–ª—è –≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω—ã—Ö –æ—Ç—Å—Ç—É–ø–æ–≤ (–≤–≤–µ—Ä—Ö/–≤–Ω–∏–∑) */

```
 
- –∑–∞–¥–∞—ë—Ç –≤–µ–ª–∏—á–∏–Ω—É –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–≥–æ –æ—Ç—Å—Ç—É–ø–∞ –æ—Ç **–∫—Ä–∞—è —ç–ª–µ–º–µ–Ω—Ç–∞ (border)** –¥–æ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–≥–æ —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ **(content)**
- –∑–Ω–∞—á–µ–Ω–∏—è –Ω–µ –º–æ–≥—É—Ç –±—ã—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º–∏
- –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –≤—Ö–æ–¥–∏—Ç –≤ –æ–±—â—É—é —à–∏—Ä–∏–Ω—É/–≤—ã—Å–æ—Ç—É —ç–ª–µ–º–µ–Ω—Ç–∞, –µ—Å–ª–∏ –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `box-sizing: border-box;`    
- –ü–∞—Ç—Ç–µ—Ä–Ω: 
		1. `padding: x` - –æ—Ç—Å—Ç—É–ø—ã —Å–æ –≤—Å–µ—Ö —Å—Ç–æ—Ä–æ–Ω
		2. `padding: x x` -  –≤–µ—Ä—Ö/–Ω–∏–∑, –ø—Ä–∞–≤–æ/–ª–µ–≤–æ
		3. `padding: x x x` - –≤–µ—Ä—Ö, –ø—Ä–∞–≤–æ/–ª–µ–≤–æ, –Ω–∏–∑
		4. `padding: x x x x` - –≤–µ—Ä—Ö, –ø—Ä–∞–≤–æ, –Ω–∏–∑, –ª–µ–≤–æ

# `position`, `top`, `left`, `bottom`, `right`, `inset`


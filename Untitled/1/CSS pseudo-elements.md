- псевдоэлементы - элементы, который не существуют в хтмл разметке и создаются через ксс с помощью спецальных селекторов (через спец. селектор можем получить досутп к какйто сущности - нпр placeholder)
- `::before & ::after` - унивесральные ксс селекторы, позовляют прицепить к правктически любому элмементу на страницу, вспомогательный элемент. исопльзуются в весртке очень часто
- `::placeholder` - это текст подсказка у полей `input` или `textarea`, который можно прописывать через аттрибут в хтмл разметке (этот текст будет отобьрадаться в поле до тех пор, пока пользователь не введет какое-то значение). позволяет получить доступ к этому элементу для стилизации. на placeholder ток действуют параметры текста и фона, в большинстве случаев от placeholderа требуется ток поменять цвет текст
  ```css
  /* стилизуем все p;laceholder`ы на странице*/
  ::placeholder{
  color: red;
  background-color: lightgray;
  }
  ```

- `::file-selector-button` - у поля прикрепления файла, т.е. input с type="file", по умолчанию есть кнопка с предложением выбрать файл. позволяет получить к ней доступ
  ```css
  input::file-selector-button{
  padding: 12px;
  color: red;
  background-color: yellow;
  border: 3px dashed black;
  border-radius: 10px;
  }
  ```
  - `::first-letter` - первая буква, любого элемента с текстовым содердимым
  ```css
  .title::first-letter{
  margin-right: 15px;
  padding: 12px 24px;
  font-size: 3rem;
  border: 3px solid skyblue;
  background-color: blue;
  box-shadow: 0 0 10px 0 rgb(0 0 0 / 0.75);
  }
  ```
  - `::first-line` - певрая строка любого элемента с текстовым содержимым. Нужно внимательно смотреть на то, какую ширину занимает блок с текстом, лучше сразщу ограчнить блок чтобы точно знать что к певрой строке применятся стили (можно также черезщ br - но костыль)
  ```html
  <p>
  Lorem ipsum dolor sit amet
  Lorem ipsum dolor sit amet
  Lorem ipsum dolor sit amet
  </p>
  ```
  ```css
  p::first-line{
  text-transform: uppercase;
  color: red;
  background-color: lightred;
  box-shadow: 0 0 10px 0 rgb(0 0 0 / 0.75);
  }
  ```
  - `::selection` - сотояние текста при его выделении. в реальной практике зачастую меняют цвет текст и фона
  ```css
  p::selection{
  color: orange;
  background-color: coral;
  text-shadow: 2px 2px 2px black;
  text-decoration: underlkine;
  }
  ```
- `::marker` - маркеры у элементов списков 
```css
li::marker{
color: red; /* цвет маркера */
}

/* or */
li::marker{
content: "o"; /* заменяем оригианльный маркер кружка */
}
```

- `::before и ::after` - позуволяет сгенерировать кастоный псевдоэлемент и прицепить его к кактому-то существующему в разметке. before - добалвяет элемент в начало, а after  - в конец. эти элементы по умолчани. являются строчными. подобных псевдоэлементов у элемента мождет быть максимум 2. чтобьы элемент появлился, ему обязщательно нужно пропаисывать св-во `content`, в котором можно прописать множество значений  (нпр пустая строка, стркоа с какимто эмодз, текстом, url до картинки - content: url('fdgdfg'), значение текущего аттриубта у текущего элемента из хтмл разметки - content: attr(data-example), тип кавычек - content: open-quote; а также значения счетчиков нумерованных список и тд). в ьбольшинстве случаем в св-ве content рпописывают пустую строку в кавычках и добавляют другие ксс свойства ради которых элемент и создается + ДОБАВИТЬ ПРИМЕРЫ

# Форматы изображений / Графика

- векторная графика - изображение созданное на основе множества точек, объединенных математическими формулами 
	- svg (scalable vector graphics) - не ткоряет качество при масташбирвоании, весит гораздно меньше чем ратсровая гарфика
- растровая графика - ?
- svg - иконки, логотипы, разные дизайнерские украшения (детальки, фигуры, линии и тд)
	- почти все что рисеут дизайнер в фигма, нужно и можно выгружать в формате свг
- jpg - фоторграии, полезный для пользваотеля окнтент, напрмер изображение товара, превью к новости и подобное
- png - фотка без фона (с прозрачным)
- `<svg>` - вставка векторной графики (если не планируем ее стилизовать или анимировать, лучше восопльзовпаться тегом `<img>`)

# Основы NMP и  установка sass compiler

```bash
# установка пакета sass
npm i -g sass
# его использование
sass --watch ./scss/main.scss ./css/styles.css
```
1. 
	- `npm` - нужно установить node.js для использования комнады
	- `i` - install
	- `-g` - global
	- `sass` - название пакета
2. 
	- `./scss/main.scss` - корневой scss файл
	- `./css/styles.css` - выходной css файл

# Основы NMP и  установка sass compiler

```bash
# установка пакета sass
npm i -g sass
# его использование
sass --watch ./scss/main.scss ./css/styles.css
```
1. 
	- `npm` - нужно установить node.js для использования комнады
	- `i` - install
	- `-g` - global
	- `sass` - название пакета
2. 
	- `./scss/main.scss` - корневой scss файл
	- `./css/styles.css` - выходной css файл

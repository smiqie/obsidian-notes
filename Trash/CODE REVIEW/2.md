По сути в @font-face достаточно подключать только woff2, т.к. он поддерживается всеми современными браузерами (woff & ttf - избыточная перестраховка для старых браузеров и IE)
В классе .container --> margin: 0 auto; можно заменить на margin-inline: auto; (более современный формат для центрирования) - по сути делает то же самое
В компоненте card - логичнее использовать тег <img>, вместо background-image. Т.к. изображение несет в себе смысловую нагрузку для пользователя (т.е. полезный контент), через background-image обычно подключается только декоративные элементы, которые не несут в себе смысла и пользы для клиента
Хорошим тоном является добавлять интерактивным элементам (в данном случае ссылкам), глобальный аттрибут title="" - который служит подсказкой для пользователя при наведении
Изображениям лучше задавать display: block; (нпр. в base.css - зависит, где сбрасываются стили. можно и в normalize.css или reset.css) - т.к. изображение по умолчанию строчный элемент и из-за этого часто под изображение появляется маленький отсуп - этот код его убирает, делая изображение блочным элементом с логическим поведением
Цвета в проекте можно вынести в глобальные перменные (с помощью :root селектора) - делается это почти в самом начале файла, и потом переиспользовать их с помощью функции var()
У цветов, вместо hex-кода, лучше использовать hsl формат (HSL - более профессиональный способ задания цвета, поэтому лучше использовать его)
На 16й строке (index.html) --> `<h1 class="visually-hidden">Restoapp</h1>`
visually-hidden избыточен, т.к. у изображение уже есть аттрибут alt="". По сути alt="restaurant logo", лучше заменить на alt="Resto App" или еще лучше alt="Resto App - Домашняя страница" (больше пользы для клиента) и он будет выполнять фактически одно и тоже
если и скрывать что-то, то лучше подойдет `<span>` вместо `<h1>` - использовать заголовок первого уровня для простой подписи к логотипу - семантически неверно
итоговый вариант:
```css 
<a class="page__logo logo" href="./index.html" title="Resto App">
        <img class="logo__image" src="./images/logo.webp" width="137" height="22" alt="Resto App - Домашняя страница">
</a>
```

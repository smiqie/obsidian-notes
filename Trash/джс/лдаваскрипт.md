
- комменты: // или /\*\*/ - хороший код не нуждлается в комментариях! лрытный разрабы их редко юзают
- let, var, const
  `let userName = 'Mark'` - можем стаивть или не стаивть тчоку с запсятой (на выбор)




- оператор typeof - похволяет определить тип перменной (мождет быть полезен с ложных преоктах)
  const someName = 'James'; 
  console.log(typeof someName);
  может бтыть полезен для отладки чужого кода
- BigInt - всроенный объект позволяющий создавать больше числа 
  Number.MAX_SAFE_INTEGER - свойство возвращающее максимальное значение Number типа, елси нужно число больше жэтого - используем BigInt // console.log(4385769843265874326587432687564328756874326528743265n) - используем BigInt вместо Number

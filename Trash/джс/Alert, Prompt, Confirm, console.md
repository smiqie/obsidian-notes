
  
# Alert, Prompt, Confirm

⚠️ Редко используются на практике

### Alert

- Показывает модальное окно с сообщением
- Останавливает код до нажатия ОК

```js
alert('Привет!');
```

**Параметры:**

- `message` - текст сообщения

---

### Prompt

- Показывает окно с полем ввода
- Возвращает `string` с введённой строкой, или `null` если ничего не введено
- Останавливает код до нажатия ОК/Отмена

```js
const name = prompt('Введите имя:');
const age = +prompt('Введите возраст:', '18'); // со значением по умолчанию
```

**Параметры:**

- `message` - текст вопроса
- `default` - значение по умолчанию (необязательно)

---

### Confirm

- Показывает окно с кнопками ОК/Отмена
- Возвращает `true` если нажали ОК или `false` если нажали Отмена
- Останавливает код до нажатия ОК/Отмена

```js
const isConfirmed = confirm('Вы уверены?');

if (isConfirmed) {
    console.log('Подтверждено');
} else {
    console.log('Отменено');
}
```

**Параметры:**

- `message` - текст вопроса

---

## 15. Console.log

- Выводит данные в консоль браузера. Используется для отладки

```js
console.log('текст'); // пример
```

- Консоль выводы мы модем группирвать

```js
console.log('Внешний файл подключен');

console.group('Группа логов'); // пример
console.log('1');
console.log('2');
console.log('3');
console.groupEnd(); // пример, конец группы логов
```

- Сообщение об ошибки (красная икнока в консоли)

```js
console.error('это ошибка');
```

- Сообщение предпрждения (желтая икнока в у сообщения)

```js
console.warn('это ошибка');
```

- создание таблицы

```js
console.table({name: "Alexsey", age: 13});
```

- создание консоли со стилями

```js
const styles = "background-color: blue; color: black; padding: 20px"; // перемнная с ксс стилями

console.log("%cStart coding, now!", styles); // %c - означеат что вторым параметром мы добавим стили? - редко пригодится но прикольная возможность // помимо %c есть еще множество других спецификаторов
```

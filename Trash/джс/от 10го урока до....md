# 11 объект Math

- встречается часто в разработке
- math - глобальный объект
- Методы/свойства (пример: `let x;`):
	-  `x = Math.min(4, 7, 2, 13) // 2;` min(переданные числа) - возвращает наименьшее из них, метод max() также но наоборот - максимальное знаечение
	- `x = Math.ceil(5.3); // передаем число с плаваюзей тчокой // вернет 6` ceil() округляет переданное дробное число в большую сторону, метод floor() полностю противвоположный, коругляет в меньшую сторону (т.е. в этом примере вернул бы 5). и есть более гибкий метод round() который округляет в блоидайшую сторону (вернул бы 5)
	- `x = Math.trunc(3.324) // 3` возвращает целую чатсь числа путем удаления дробных знаков
	- `x = Math.pow(3, 4) // 81` pow() возвводит в степень, 1й аргумент - само число которой возводим, второй аргумент - степень которая применяется к числу
	- `x = Math.sqrt(81) // 9` в отличие от pow() делает обратную операцию, возвращает квадртаный корень
	- `x = Math.abs(-48) // 48` abs() принимает аргументом само число и возвращает его абсолютное знаечние (модуль)
	- `x = Math.random()` возвращает рандомное число (дробное целое без разницы)
		- `Math.floor(random() * 50 + 1)` удучшение логики.комбинация. floor коругляем число к меньшему + полученное число умножаем на 50 + 1 чтобы получить более красивое число (в этом примере мы генерировали числа от 1 до 50ти). по сути нужно методл рандом умнождить на то число до которого мы хотим генерировать числа (включая его или не?) а единичку прибавляем чтобы отчет наичнался с красивого 1? // получаем рандомные числа в пределах 50ти

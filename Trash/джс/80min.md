
## 11. Условные конструкции

### If/Else

```js
if (условие) { // 1я ветка
    // код
}
 else if (другое_условие) { // 2я ветка
    // код
} 
else { // доп ветка
    // код
}
```

- все что просиходит в окнструкции if, в скобках - преобразумется к логическому типу (число 0, пустая строка "", NaN, null, undefined - становятся false (значения окторые преобразуются в false называются falsy values),  а значения окторые преобразуются в true называются  thruthy values (правдивые и ложные значения)
	- могут српосить на собесе
	- ```js
	  if(0){} // false / falsy value
	  if(1){} // true / thuthy value

	  if(""){} // false / falsy value
	  if(" "){} // true / thuthy value
	  
	  etc
	  ```

**❌ Плохой код (хуже читается):**

```js
if (условие)
    // одна строка кода без фигурных скобок
```

**✅ Хорошо - всегда используйте фигурные скобки**

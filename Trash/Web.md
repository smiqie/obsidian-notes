- **CSS** (Cascading Style Sheets) - оформление и внешний вид страницы
- **HTML**  (HyperText Markup Language) - структура страницы

- # 🏗️Виды сайтов
- **Landing page (лендинг)** - одностраничный сайт (нпр. рекламный или презентационный)
- **Многостраничный сайт** - содержит несколько связанных страниц (нпр. интернет-магазин, блог, форум)
- **Веб-приложения** - программы, работающие в браузере (нпр. Gmail, Figma, Google Docs)
    Работают на стороне сервера, т.е. вычисления происходят не на компьютере пользователя, а на сервере → экономит ресурсы устройства\

# 🚀SEO
- **SEO** (Search Engine Optimization) - поисковая оптимизация; набор методов, помогающих сайту подниматься выше в поисковых системах (Google, Yandex и т.д.)

# 🧩 Этапы создания сайта
1. **Макет** - дизайн сайта (нпр., _Figma_)
2. **Верстка (Frontend)** - реализация внешнего вида (HTML, CSS, JavaScript)
3. **Backend** - серверная часть, логика и хранение данных (PHP, Python, Node.js и тд.)

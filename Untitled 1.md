# String

# Создание строк

## Кавычки

```js
'text'                  // одинарные
"text"                  // двойные
`text ${variable}`      // обратные кавычки (backticks) - интерполяция
```

**Обратные кавычки сохраняют форматирование:**

```js
let str = `User:
- Alex
- Max
- Julia
`;
```

---

## Интерполяция (шаблонные строки) и Конкатенация

```js
const name = 'Max';
const a = 5, b = 3;

const msg = `name: ${name}, sum: ${a + b}`; // ✅ читабельно

// ❌ Плохой тон - конкатенация через +
const sum = "sum: " + (a + b);
```

# Escape-последовательности

- `\n` - перенос на новую строку
- `\"` - двойная кавычка
- `\'` - одинарная кавычка
- `\\` - обратный слэш
- `\t` - табуляция
- `\uXXXX` - unicode символ (4 шестнадцатеричных символа)

# Спецсимволы и эмодзи

## Unicode символы (короткие коды)

```js
const copyRight = "\u00A9"; // ©
const euro = "\u20AC";      // €
```

- `\uXXXX` - 4 шестнадцатеричных символа
	- вставка символов по короткому коду (до 4 символов)

## Unicode символы (длинные коды) - эмодзи

```js
// ❌ Не работает (код слишком длинный)
const thumbsUp = "\u1F44D";

// ✅ Работает (фигурные скобки для длинных кодов)
const thumbsUp = "\u{1F44D}"; // 👍
```

- `\u{XXXXX}` - любое количество шестнадцатеричных символов в фигурных скобках
- **Важно:** фигурные скобки `{}` нужны для кодов **длиннее 4 символов** (например, эмодзи)

---

## Эмодзи - способы вставки

### 1. Напрямую

```js
const emoji = "🥵";
```

**Самый простой способ** - просто скопировать эмодзи

---

### 2. Через Unicode с фигурными скобками

```js
const thumbsUp = "\u{1F44D}"; // 👍
const fire = "\u{1F525}";     // 🔥
```

**Параметры:** `\u{код}` - Unicode код в фигурных скобках **Когда использовать:** когда нужно вставить эмодзи по коду

---

### 3. `String.fromCodePoint()` - из кода Unicode

```js
String.fromCodePoint(128077); // 👍
String.fromCodePoint(128293); // 💥
```

**Параметры:** `codePoint` - десятичный код Unicode (число) **Возвращает:** символ (эмодзи) **Назначение:** создание символа из десятичного кода

**Разница между способами 2 и 3:**

- `"\u{1F44D}"` - шестнадцатеричный код (0x1F44D)
- `String.fromCodePoint(128077)` - десятичный код (128077 = 0x1F44D)

# `String.raw()` - сохранение пути к файлу

```js
const filePath = String.raw`C:\Development\profile\aboutme.html`;
```

**Параметры:** шаблонная строка **Возвращает:** строку без обработки escape-последовательностей **Назначение:** сохранение путей к файлам (не обрабатывает `\`)

# Свойства строки

## `.length` - длина строки

```js
const text = "World";

text.length; // 5
```

**Параметры:** нет **Возвращает:** число (количество символов в строке)

---

# Доступ к символам

## По индексу `[n]`

```js
const text = "World";

text[0]; // 'W' (первый символ)
text[4]; // 'd' (последний символ)
```

**Параметры:** индекс (от 0) **Возвращает:** символ или `undefined`

---

## `.at(index)` - получить символ по индексу

```js
const text = "World";

text.at(0);  // 'W'
text.at(4);  // 'd'
text.at(-1); // 'd' (с конца)
text.at(-2); // 'l' (предпоследний)
```

**Параметры:** `index` - индекс символа (может быть отрицательным) **Возвращает:** символ **Особенность:** поддерживает отрицательные индексы (отсчет с конца)

---

## `.charAt(index)` - получить символ

```js
const text = "World";

text.charAt(0); // 'W'
text.charAt(4); // 'd'
```

**Параметры:** `index` - индекс символа **Возвращает:** символ **Особенность:** НЕ поддерживает отрицательные индексы

---

# Методы изменения регистра

## `.toUpperCase()` - к верхнему регистру

```js
const text = "World";

text.toUpperCase(); // 'WORLD'
```

**Параметры:** нет **Возвращает:** новую строку (все буквы в верхнем регистре)

---

## `.toLowerCase()` - к нижнему регистру

```js
const text = "World";

text.toLowerCase(); // 'world'
```

**Параметры:** нет **Возвращает:** новую строку (все буквы в нижнем регистре)

---

# Методы работы с пробелами

## `.trim()` - убрать пробелы с обеих сторон

```js
const str = "   TEXT   ";

str.trim(); // 'TEXT'
```

**Параметры:** нет **Возвращает:** строку без пробелов с начала и конца **Назначение:** используется чаще всего (проверка input полей)

---

## `.trimStart()` - убрать пробелы слева

```js
const str = "   TEXT   ";

str.trimStart(); // 'TEXT   '
```

**Параметры:** нет **Возвращает:** строку без пробелов с начала

---

## `.trimEnd()` - убрать пробелы справа

```js
const str = "   TEXT   ";

str.trimEnd(); // '   TEXT'
```

**Параметры:** нет **Возвращает:** строку без пробелов с конца

---

# Методы повторения и дополнения

## `.repeat(count)` - повторить строку

```js
const text = "World";

text.repeat(3); // 'WorldWorldWorld'
```

**Параметры:** `count` - количество повторений **Возвращает:** новую строку (повторенную n раз)

---

## `.padStart(length, fillString)` - дополнить слева

```js
const str = "abc";

str.padStart(10, "0"); // '0000000abc'
str.padStart(5, "x");  // 'xxabc'
```

**Параметры:**

- `length` - итоговая длина строки
- `fillString` - символ для заполнения

**Возвращает:** новую строку (дополненную слева)

---

## `.padEnd(length, fillString)` - дополнить справа

```js
const str = "abc";

str.padEnd(10, "0"); // 'abc0000000'
str.padEnd(5, "x");  // 'abcxx'
```

**Параметры:**

- `length` - итоговая длина строки
- `fillString` - символ для заполнения

**Возвращает:** новую строку (дополненную справа)

---

# Методы замены

## `.replace(searchValue, replaceValue)` - заменить

```js
const str = "JavaScript is awesome";

str.replace("Java", "Type"); // 'TypeScript is awesome'
```

**Параметры:**

- `searchValue` - что ищем (строка)
- `replaceValue` - на что заменяем (строка)

**Возвращает:** новую строку (с заменой **первого** вхождения)

---

# Методы поиска

## `.indexOf(searchString)` - индекс первого вхождения

```js
const str = "JavaScript is awesome";

str.indexOf("Script"); // 4
str.indexOf("Python"); // -1 (не найдено)
```

**Параметры:** `searchString` - что ищем **Возвращает:** индекс первого вхождения или `-1` если не найдено

**Правильная работа с indexOf:**

```js
const str = "JavaScript is awesome";

// ❌ Неправильно
if (str.indexOf("Java")) { // 0 - falsy value
    console.log('Найдено');
}

// ✅ Правильно (всегда сравнивать с -1)
if (str.indexOf("Java") !== -1) { // 0 !== -1 - true
    console.log('Найдено');
}
```

---

## `.includes(searchString)` - проверка вхождения

```js
const str = "JavaScript is awesome";

str.includes("is");     // true
str.includes("Python"); // false
```

**Параметры:** `searchString` - что ищем **Возвращает:** `true` если найдено, `false` если нет **Назначение:** используется часто (проще чем indexOf)

---

## `.startsWith(searchString)` - начинается ли строка

```js
const str = "JavaScript is awesome";

str.startsWith("Java");   // true
str.startsWith("Script"); // false
```

**Параметры:** `searchString` - с чего должна начинаться строка **Возвращает:** `true` или `false`

---

## `.endsWith(searchString)` - заканчивается ли строка

```js
const str = "JavaScript is awesome";

str.endsWith("awesome"); // true
str.endsWith("Java");    // false
```

**Параметры:** `searchString` - на что должна заканчиваться строка **Возвращает:** `true` или `false`

---

# Методы извлечения подстроки

## `.substring(start, end)` - извлечь подстроку

```js
const str = "hardcore";

str.substring(0, 4); // 'hard' (с 0 до 4, не включая 4)
str.substring(4);    // 'core' (с 4 до конца)
str.substring(8, 4); // 'core' (меняет местами если start > end)
```

**Параметры:**

- `start` - индекс начала
- `end` - индекс конца (не включительно, необязательный)

**Возвращает:** подстроку **Особенность:** если `start > end` - меняет их местами

---

## `.slice(start, end)` - извлечь подстроку

```js
const str = "hardcore";

str.slice(0, 4); // 'hard' (с 0 до 4, не включая 4)
str.slice(4);    // 'core' (с 4 до конца)
str.slice(8, 4); // '' (пустая строка, НЕ меняет местами)
str.slice(-4);   // 'core' (последние 4 символа)
```

**Параметры:**

- `start` - индекс начала (может быть отрицательным)
- `end` - индекс конца (не включительно, необязательный)

**Возвращает:** подстроку **Особенность:** поддерживает отрицательные индексы, НЕ меняет местами start/end

---

## `.substr(start, length)` - ❌ УСТАРЕВШИЙ

```js
const str = "hardcore";

str.substr(0, 4); // 'hard'
```

**НЕ ИСПОЛЬЗОВАТЬ** - устаревший метод

---

# Разница между substring и slice

|Метод|Отрицательные индексы|Если start > end|
|---|---|---|
|`substring`|❌ не поддерживает|меняет местами|
|`slice`|✅ поддерживает|возвращает пустую строку|

**Рекомендация:** использовать `.slice()` - более гибкий

# `background-color` - цвет фона

```css
background-color: lightblue;
background-color: #ff0000;
background-color: rgb(255, 0, 0);
```

- отвечает только за цвет фона элемента

---

# `background-image` - фоновое изображение

```css
/* Изображение */
background-image: url('img/bg.jpg');

/* Градиент */
background-image: linear-gradient(45deg, red, blue);
background-image: conic-gradient(red, blue);

/* Несколько изображений (наложение друг на друга) */
background-image:
  url('./cat-1.png'), /* будет в самом верху */
  url('./cat-2.png'),
  url('./cat-3.png'); /* будет в самом низу */
```

- задает фоновое изображение или градиент элементу
- `url()` - путь к изображению (относительный или абсолютный)
- позволяет накладывать изображения друг на друга (например, с прозрачным фоном)

---

# `background-position` - позиция фона

```css
background-position: center;
background-position: top right;
background-position: 50% 50%;
background-position: 100px 200px;
```

- задает позицию фонового изображения относительно контейнера
- значение по умолчанию: `0 0`
- значения:
	1. `center` - **самое популярное значение**, `top`, `bottom`, `left`, `right`
	2. `px` 
	3.  `%` (обычно указывают в `%` или `px`)
- **Паттерны:**
    1. `background-position: X Y;` - **X:** горизонталь (слева → справа), **Y:** вертикаль (сверху → вниз)
        - если указано только одно значение, то позиционирование будет настроено только по горизонтали
    2. `background-position: <side> <offset> <side> <offset>;`
	    - **side:** сама сторона
	    - **offset:** отступ от неё

---

# `background-repeat` - повторение фона

```css
background-repeat: no-repeat; 
background-repeat: repeat-x;
background-repeat: space;
```

- задает, будет ли фон повторяться на все свободное место контейнера
- **Значения:**
	- `no-repeat` - изображение не повторяется
    - `repeat` - повторяется по обеим осям (по умолчанию)
    - `repeat-x` - повторяется по горизонтали
    - `repeat-y` - повторяется по вертикали
    - `space` - повторяется с равными промежутками по обеим осям, заполняя всё пространство контейнера
    - `round` - повторяется  с растягиванием по обеим осям, для заполнения всего пространства контейнера

---

# `background-size` - размер фона

```css
background-size: cover;
background-size: contain;

background-size: 100% 200px;
background-size: 100px auto;   /* ширина 100px, высота пропорционально */

background-size: auto;        /* оригинальный размер */
```

- задает размер фонового изображения относительно контейнера
- **Значения:**
	1. `cover` - картинка заполняет весь блок, сохраняя пропорции (может обрезаться, если не помещается)
		- идеально для фонов на весь экран (самое популярное значение)
	2. `contain` - картинка полностью помещается в блок, сохраняя пропорции (в контейнере могут появиться пустые места)
		- если нужно показать всю картинку целиком
	3. `точные размеры (ширина × высота)` - в любых единицах измерения
		- при задании ширины и высоты пропорции могут искажаться, чтобы сохранить пропорции, для одной из сторон указать `auto` 
	4. `auto` - оригинальный размер (значение по умолчанию)

---

# `background-attachment` - поведение при скролле

```css
background-attachment: fixed; 
```

- задает правила фона при скролле (будет ли фон прокручиваться вместе со страницей)
- **Значения:**
	1. `scroll` - фон прокручивается вместе со страницей (по умолчанию)
	2. `fixed` - фон закреплён относительно окна браузера и остаётся неподвижным при прокрутке страницы
	3. `local` - фон привязан к области прокрутки элемента и двигается вместе с его скроллом

# `background` - shorthand для всех background-свойств

```css
background: lightblue url('img/bg.jpg') center / cover no-repeat;
```

- сокращенная запись для всех `background-` свойств
- **НЕ рекомендуется** использовать для установки только цвета (`background: red;`) - лучше `background-color: red;`
    - `background` использует множество подсвойств, и этой записью можно их всех переопределить

# `linear-gradient()` - линейный градиент

```css
background: linear-gradient(45deg, red, blue);
background: linear-gradient(to right, red, blue);
background: linear-gradient(red 40%, green 50%, blue);
```

- создает линейный градиент (плавный переход между цветами)
- **Паттерн:** `linear-gradient(направление, цвет-1, цвет-2, ...)`
- **Направление:**
    - угол: `45deg`, `90deg`, `180deg` (по умолчанию) и т.д. (от 0 до 360)
    - turn единицы: `1turn = 360deg`, `0.5turn = 180deg`
    - ключевые слова: `to top` (0deg), `to bottom` (180deg), `to left` (270deg), `to right` (90deg), `to top right` (45deg) и т.д.
- **Точки остановки (color stops):** ^colorStops
    - 1-я точка - при котором закончится плавный переход из предыдущего цвета
    - 2-я точка - с которого начнется плавный переход в следующий цвет **и т.д.**
    - пример: `linear-gradient(blue 30%, red 40% 60%, green 90%)`
		- проценты считаются от высоты элемента

# `radial-gradient()` - радиальный градиент

```css
background: radial-gradient(circle, red, blue);
background: radial-gradient(red 30%, blue 70%);
```

- создает радиальный градиент (от центра к краям)
- **Паттерн:** `radial-gradient(форма, цвет-1, цвет-2, ...)`
- **Форма:** `circle` (круг) или `ellipse` (эллипс, по умолчанию)
- **Точки остановки (color stops):**  [[Visual Appearance (Design)#^colorStops]]

# `conic-gradient()` - конический градиент

```css
/* Простой градиент — от красного к синему по кругу, начиная сверху (0deg) */
background: conic-gradient(red, blue);

/* Изменяем начало градиента с 0° (сверху) на 45° */
background: conic-gradient(from 45deg, red, blue);

/* Изменяем центр градиента: по умолчанию центр блока (50% 50%) */
background: conic-gradient(at 30% 70%, red, blue);

/* Сочетание: начало с 90° и центр в левом верхнем углу */
background: conic-gradient(from 90deg at 0% 0%, red, blue);
```

- создает конический градиент (градиент **по кругу вокруг центра**)
- **Паттерн:** `conic-gradient(цвет-1, цвет-2, ...)`
- **Начало градиента (`from`):** по умолчанию 0deg(сверху)
	- можно сместить через `from` (`x°deg`)
	- угол считается по часовой стрелке
		1. `0deg` - сверху
		2. `90deg` - справа
		3. `180deg` - снизу
		4. `270deg` - слева
- **Центр градиента (`at`):** по умолчанию - центр элемента (`50% 50%`)
	- можно указать любую точку внутри блока (`x% y%`)
- **Точки остановки (color stops):** 
    - пример: `conic-gradient(red 0deg 90deg, blue 90deg 180deg)`
	    - красный от 0° до 90°, синий от 90° до 180°, дальше переходы к остальным цветам

# `border` - граница элемента

```css
border: none;   /* убрать границу */

/* Отдельные стороны */
border-top: 2px solid red;
border-right: 1px dashed green;
border-bottom: 3px dotted blue;
border-left: 4px double purple;
border-block: 10px solid blue;   /* вертикаль */
border-inline: 10px solid blue;  /* горизонталь */

/* Отдельные стороны, но через подсвойства (верхняя сторона): */
border-top-width: 2px;
border-top-style: dashed;
border-top-color: red;

/* shorthand: width style color */
border: 1px solid black;         

/* Отдельные свойства */
border-width: 2px;
border-style: solid;
border-color: black;
```

- `border: none;` рекомендуется прописывать перед добавлением своей рамки - убирает дефолтные границы

- **`border-width` паттерн:**
	1. `1 значение` - все стороны одинаковые
	2. `2 значения` - верх/низ, право/лево
	3. `3 значения` - верх, право/лево, низ
	4. `4 значения` - верх, право, низ, лево

- Можно задавать `border-width` с помощью ключевых слов `thin, medium, thick`
	- **крайне не популярный формат, в большинстве случаев используют px**

- **`border-style` значения:** ^borderStyle
	1. `none` - без границы (по умолчанию)
	2. `solid` - сплошная **(самое популярное, остальные используют крайне редко)**
	3. `dashed` - пунктир
	4. `dotted` - точки
	5. `double` - двойная

- **Важно:** для каждой стороны можно отдельно задавать `width`, `style` и `color`

---

# `border-radius` - скругление углов

```css
border-radius: 10px;             /* все углы */
border-radius: 50%;              /* круг */
border-radius: 0 30px 80% 30px;  /* интересные фигуры */

/* Отдельные углы */
border-top-left-radius: 5px;
border-top-right-radius: 10px;
border-bottom-left-radius: 15px;
border-bottom-right-radius: 20px;
```

- скругление углов у элемента **(НЕ имеет отношение к свойству `border`!)**
- отлично работает со свойством `outline`
- можно задавать в любых единицах измерения, но **обычно используют `px` или `%`**

- **Паттерн:**
	1. `1 значение` - все углы одинаковы
	2. `2 значения` - верхние / нижние
	3. `3 значения` - верхний левый, верхний правый + нижний левый, нижний правый
	4. `4 значения` - верхний левый, верхний правый, нижний правый, нижний левый
		- **можем получить довольно интересные фигуры**

# `outline` - контур элемента

```css
outline: 2px solid red;    /* shorthand: width style color */
outline: none;             /* убрать outline */

/* Отдельные свойства */
outline-width: 2px;
outline-style: solid;
outline-color: blue;

outline-offset: 5px;       /* смещение от границы */
```

- создает контур (обводку) вокруг элемента **со всех сторон**
	- **НЕ влияет на фактические размеры** элемента (в отличие от `border`)
- `outline-offset` - расстояние между контуром и границей элемента (может быть отрицательным - контур внутрь)
	- значение по умолчанию: `0`
	- можно задавать в любых единицах измерения, но **обычно используют px**
- значения `outline-style`: [[Visual Appearance (Design)#^borderStyle]]
- чаще используется для **фокуса** на интерактивных элементах (кнопки, ссылки, инпуты)

# `box-shadow` - тень элемента

```css
/* Паттерн: x y blur spread color */
box-shadow: 10px 10px 5px 0px rgb(0 0 0 / 0.5);

/* Несколько теней */
box-shadow: 
  5px 5px 10px red,
  -5px -5px 10px blue;

/* Внутренняя тень */
box-shadow: inset 5px 5px 10px rgba(0, 0, 0, 0.3);

/* Убрать тень */
box-shadow: none;
```

**Параметры:**

1. **x** (horizontal offset) - смещение по горизонтали (`положительное` → вправо, `отрицательное` → влево)
2. **y** (vertical offset) - смещение по вертикали (`положительное` → вниз, `отрицательное` → вверх)
3. **blur** (blur radius) - радиус размытия (`0` = четкая тень (по умолчанию), чем больше - тем размытее)
4. **spread** (spread radius) - растяжение тени (`положительное` → тень больше элемента, `отрицательное` → тень меньше элемента, `0` - тень размером с элемент)
5. **color** - цвет тени
6. **inset** - делает тень внутренней (добавляется в начало)

- `box-shadow` не влияет на размеры элемента и не занимает место в потоке
- **для указания значений чаще используют `px`**

# `filter` - графические эффекты

```css
filter: none; /* без фильтров (значение по умолчанию) */

filter: blur(5px);               /* размытие */
filter: brightness(1.5);         /* яркость */
filter: contrast(200%);          /* контрастность */
filter: grayscale(100%);         /* черно-белое */
filter: invert(1);               /* инверсия */
filter: saturate(2);             /* насыщенность */
filter: hue-rotate(90deg);       /* поворот оттенка */
filter: sepia(100%);             /* эффект сепии */
filter: opacity(0.5);            /* прозрачность */

filter: drop-shadow(5px 5px 10px rgb(0 0 0 / 0.5)); /* тень с учётом прозрачности */

/* Несколько фильтров подряд (применяются слева направо) */
filter: brightness(1.2) contrast(1.1) saturate(1.3);
```

- применяет **графические эффекты** к любому элементу (картинки, текст, контейнеры)

- **Основные функции `filter`:**

	1. `blur(radius)` - размытие
		- **Значения**: 
			- **!обычно указывают в `px`!**
				- чем больше радиус - тем сильнее размытие
	2. `brightness(amount)` - яркость
		- **Значения:**
			- можно писать в формате коэффициенте и в процентах
				- `0 / 0%` - чёрный **//**  `1 / 100%` - оригинал **//**  `>1` - светлее
	3. `contrast(amount)` - контрастность
		- **Значения:**
			- можно писать в формате коэффициенте и в процентах
				- `0 / 0%` - плоское серое пятно **//** `1 / 100%` - оригинал **//** `>1` - усиленный контраст
	4. `grayscale(amount)` - оттенки серого
		- **Значения:**
			- можно писать в формате коэффициенте и в процентах
				- `0 / 0%` - цветное (по умолчанию) **//** `1 / 100%` - полностью чёрно-белое
	5. `invert(amount)` - инверсия цветов
		- **Значения:**
			- можно писать в формате коэффициенте и в процентах
				- `0 / 0%` - нормальные цвета (по умолчанию) **//** `1` / `100%` - полная инверсия
	
	6. `saturate(amount)` - насыщенность
		- **Значения:**
			- можно писать в формате коэффициенте
				- `0 / 0%` - полностью обесцвечено **//** `1 / 100%` - оригинал **//** `>1` - яркие, насыщенные цвета
	
	7. `hue-rotate(angle)` - поворот цветового круга (поворот цветового круга на `x°`)
		- **Значения:** 
			- в градусах: `deg`
				- `0deg` - оригинал **//** `360deg` - полный круг, вернётся к исходному состоянию (снова оригинал)
	
	8. `sepia(amount)` - сепия (эффект старого фото)
		- **Значения:**
			- можно писать в формате коэффициенте и в процентах
				- `0 / 0%` - без изменений **//** `1 / 100%` - максимальная сепия
	
	9. `opacity(amount)` - прозрачность
		- **Значения:**
			- можно писать в формате коэффициенте и в процентах
				- `0 / 0%` - полностью прозрачно **//** `1 / 100%` - непрозрачно (по умолчанию)
		- **Интересный факт:** работает чуть быстрее обычного `opacity` за счёт  аппаратного ускорения (разница ничтожна)
			- Аппаратное ускорение -  это когда браузер передаёт обработку графики видеокарте (GPU), а не центральному процессору (CPU), которая делает такие операции быстрее, чем процессор
	
	10. `drop-shadow(x y blur color)` - тень, учитывающая форму объекта
		- **Значение по умолчанию:** `none`
		- в отличие от `box-shadow`, форму элемента (прозрачные области), поэтому идеально подходит для PNG **(частый приём)**
		- **паттерн:** `drop-shadow(offset-x offset-y blur-radius color)`

**Примеры:**

```css
/* Эффект "старое фото" */
.vintage { filter: sepia(80%) brightness(0.9) contrast(1.1); }

/* Размытый фон */
.blur-background { filter: blur(10px); }

/* Неоновый эффект */
.neon:hover { filter: brightness(1.5) saturate(2) hue-rotate(45deg); }


/* чёрно-белое изображение */
.bw-image { filter: grayscale(100%); }

/* неоновый эффект при наведении */
.neon:hover { filter: brightness(1.5) saturate(2) hue-rotate(45deg); }

/* эффект свечения */
.glow { filter: brightness(1.3) drop-shadow(0 0 20px rgb(255 255 255 / 0.8)); }

/* размытый фон (для модальных окон) */
.backdrop { filter: blur(8px) brightness(0.7); }

/* неоновый текст */
.neon-text {
  filter: drop-shadow(0 0 10px #0ff) 
          drop-shadow(0 0 20px #0ff) 
          drop-shadow(0 0 30px #0ff);
}
```

# `opacity` - прозрачность

```css
opacity: 1;    /* полностью непрозрачный (по умолчанию) */
opacity: 0.5;  /* полупрозрачный */
opacity: 0;    /* полностью прозрачный */
```

- управляет прозрачностью элемента
- значение: число от 0  до 1 (по умолчанию)
- элемент прозрачный, но с ним можно взаимодействовать

# `visibility` - видимость элемента

```css
visibility: visible; /* элемент видимый (по умолчанию) */
visibility: hidden;  /* элемент невидимый + нельзя взаимодействовать */
```

- управляет видимостью элемента + запрещает любое взаимодействие
- отличие от `opacity`: нельзя добавить плавный переход из одного состояния в другое
- **отличие от `display: none`:** элемент остается в потоке и занимает место

# `background-clip` - область распространения фона

```css
/* Градиентный текст */
.gradient-text {
  background: linear-gradient(45deg, red, blue);
  background-clip: text;
  -webkit-background-clip: text;
  color: transparent;
}

/* Изображение внутри текста */
.image-text {
  background-image: url('pattern.jpg');
  background-clip: text;
  -webkit-background-clip: text;
  color: transparent;
}
```

- определяет до какой области блочной модели распространяется фон
	- **используется редко, но знать полезно**
- **Значения:**
    - `border-box` - фон до внешнего края border (включая border) (по умолчанию)
    - `padding-box` - фон до внешнего края padding (не включает border)
    - `content-box` - фон только в области content (не включая padding и border)
    - `text` - фон обрезается по форме текста (требует `color: transparent;` и префикс `-webkit-`)
		- идеально для градиентного текста
